## Daili Code

Daili Code是一款开源的AI Agent命令行工具，作为Gemini CLI的派生版本，其核心在于提供了与多种大型语言模型（LLM）的广泛兼容性，包括Google Gemini、OpenAI以及任何遵循OpenAI API标准的自定义LLM，旨在通过命令行界面协助开发者连接各类工具、深入理解代码并显著加速工作流。该工具能够利用LLM的先进能力进行大型代码库的查询与编辑，支持大上下文窗口；通过多模态功能从PDF或草图生成全新应用；自动化复杂的开发运维任务，如处理Pull Request或rebase操作；并通过工具和MCP服务连接新功能。此外，它允许用户通过简单的环境变量灵活配置和无缝切换首选的LLM提供商，项目还展示了对多种不同LLM模型在思考过程、简单任务、工具调用、复杂任务和多模态能力等多个维度的广泛测试，证明了其强大的通用性和实用性。
