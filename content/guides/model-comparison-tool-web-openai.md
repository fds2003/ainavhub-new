## AI Model Comparison Tool

“AI模型对比工具”是一款纯前端、美观直观的Web应用程序，主要用于并排比较两个不同AI模型的响应效果。该工具的核心功能包括同时调用两个兼容OpenAI API的AI模型，支持多轮对话以保持上下文连贯，并能显示响应时间、token使用量等详细元数据。它还具备API配置自动保存（不含敏感密钥）、一键API连接测试、快捷键操作（如Ctrl+Enter启动对比）以及双击复制响应内容等便捷特性。该项目强调安全性，API密钥仅在本地使用且不被保存或传输，并且采用响应式设计，用户只需直接在浏览器中打开`index.html`文件即可开始使用，无需任何后端服务部署。
