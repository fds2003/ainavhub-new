## LightAgent

**LightAgent** 是一个极其轻量级的主动智能代理框架，具备记忆 (`mem0`)、工具 (`Tools`) 和思维树 (`ToT`) 的功能。它支持类似群体的多代理协作、自动化工具生成和代理评估，并能兼容多种大型模型，如 OpenAI、智谱 ChatGLM、百川大模型等，同时支持 OpenAI 的流式 API 服务输出，可以无缝集成到主流聊天框架中。

##### 主要特点

1. **轻量高效** ：采用极简设计，快速部署，适合多种应用场景。
2. **记忆支持** ：支持用户自定义的长期记忆，智能管理用户的个性化记忆。
3. **自主学习** ：每个代理具备独立学习能力，通过对话不断优化。
4. **工具集成** ：支持自定义工具和自动化生成的工具，灵活扩展以满足多样化需求。
5. **复杂目标处理** ：通过思维树模块支持复杂任务的分解与多步骤推理。
6. **多代理协作** ：简化多代理协作的实现，通过内置 LightSwarm 支持意图识别和任务分配。
7. **独立执行** ：任务与工具调用可独立完成，无需人工干预。
8. **多模型支持** ：兼容多种大型语言模型，支持多种服务场景。
9. **流式 API** ：支持 OpenAI 流式 API 输出，增强用户体验。
10. **工具生成器** ：用户只需提供 API 文档，即可快速生成定制工具。


##### 使用场景

* **智能客服** ：通过多轮对话和工具集成提供高效的客户支持。
* **数据分析** ：利用思维树和多代理协作处理复杂数据分析任务。
* **自动化工具** ：快速构建定制化工具以应对特定需求。
* **教育助理** ：通过记忆模块和流式 API 提供个性化学习体验。


LightAgent 提供了一个高效、灵活的框架，帮助开发者快速构建智能应用，以便满足各种需求。
