## Engy

##### Engy简介

Engy 是一款由人工智能驱动的应用程序生成器，旨在改变现代应用开发方式。在大型语言模型（LLM）引领的新时代中，Engy 的出现使得开发人员无需再从零开始编码。无论你是资深开发者还是初学者，Engy 都能帮助你将创意转化为功能齐全的网页应用，支持 Python 后端和 HTML/JavaScript 前端开发。Engy 相当于一个强大的开发团队，能实现从简单工具到复杂数据分析应用的各种需求。

##### 使用场景

Engy 可以帮助用户创建多种类型的应用，包括但不限于：

* 📝 生产力工具和任务管理器
* 🗂️ 数据组织和管理系统
* 🌐 网页爬虫和数据收集工具
* 📊 简单的数据可视化工具


##### 主要功能

1. **快速创建应用** ：用户只需描述他们的应用想法，Engy 即可自动生成相应的代码。
2. **轻松添加新功能** ：可以方便地为现有应用添加新特性，Engy 会自动实现。
3. **前后端调整** ：提供工具进行后端和前端的微调，以增强应用的功能和用户界面。
4. **重构应用** ：支持将应用的代码拆分为模块化文件，提升可维护性和可扩展性。
5. **克隆现有应用** ：用户可以基于现有应用快速创建新的应用。


##### 示例展示

Engy 附带多种示例项目，帮助用户了解其功能，例如账户管理器、PDF 合并器、网页爬虫等。此外，还有基本工具和高级示例，覆盖多种使用场景和应用需求。

##### 贡献与支持

Engy 由两位热爱工程的开发者维护，欢迎任何形式的贡献。用户若遇到问题，可以在 GitHub 上提出问题，或直接联系开发团队获取帮助。

Engy 的出现大大降低了应用开发的门槛，使得更多人能够轻松地实现他们的创意。
