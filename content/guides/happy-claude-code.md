## Happy

Happy 是一个免费开源的项目，旨在实现对 Claude Code 的移动设备远程控制和实时交互。该工具的核心功能在于用户可以通过移动设备，扫描二维码连接正在运行的 Claude Code 会话，并实现两端的实时会话共享，从而随时随地管理和控制代码。它提供了一系列核心命令，包括认证管理、启动 Codex 模式以辅助编码、在 Happy 云端安全存储 AI 供应商 API 密钥、向连接的设备发送即时推送通知、管理后台服务以及进行系统诊断。此外，Happy 支持高度定制化，允许用户指定使用的 Claude 模型、设置权限模式，并能传递额外的环境变量或参数，提供了一个灵活且高效的移动代码控制解决方案。
