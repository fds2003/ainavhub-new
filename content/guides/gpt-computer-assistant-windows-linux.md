## GPT Computer Assistant

##### GPT Computer Assistant简介

GPT Computer Assistant是一款旨在为Windows和Linux操作系统用户提供类似于ChatGPT MacOS应用程序的解决方案。它当前可以作为Python库安装，未来计划提供简便的本地安装脚本（如.exe文件）。该助手由Upsonic Tiger 🐅（一个用于llm代理的功能中心）提供支持。

###### 安装与运行


pip3 install gpt-computer-assistant


运行程序：


computerassistant


可以选择不同的配置文件，这些配置文件有各自独立的消息历史记录，例如：工作、个人、upsonic。

使用指定配置文件运行：


computerassistant --profile personal


升级程序：


pip3 install gpt-computer-assistant --upgrade


###### 功能

GPT Computer Assistant具备多种功能，包括但不限于：

* 屏幕读取
* 使用麦克风
* 系统音频管理
* 记忆功能
* 打开和关闭应用程序
* 打开URL
* 剪贴板操作
* 搜索引擎集成
* Python和Shell解释器
* 编写和运行脚本
* 使用Telegram账户
* 知识管理


计划中的功能：

* 面部分析（通过DeepFace）
* Windows本地安装文件（.exe）
* Linux本地安装文件
* MacOS本地安装文件


###### 使用场景

GPT Computer Assistant可以在以下场景中使用：

* 会议记录：通过录音并转换为文字记录会议内容。
* 日常助理：提供类似日历提醒、任务管理等功能。
* 阅读文档：帮助用户快速浏览和理解文档内容。
* 编程助手：提供代码建议、调试和运行功能等。


通过该工具，用户能够更高效地管理日常任务，提高生产力。
