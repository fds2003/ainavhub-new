## Nanocoder

Nanocoder是一个专注于隐私和控制的本地优先命令行（CLI）编程代理工具，旨在将强大的代理编程能力赋能于本地AI模型或如OpenRouter等受控API。其核心功能包括支持Ollama、OpenRouter以及其他OpenAI兼容API等多种AI提供商，并提供一个高级工具系统，通过Model Context Protocol（MCP）服务器可动态加载并执行文件操作和系统命令。此外，它允许用户通过Markdown定义可复用、带参数的自定义命令，并通过智能自动补全、会话持久化等特性显著提升了命令行用户体验。该项目还具备可扩展的TypeScript架构和项目级配置能力，为开发者提供了强大的灵活性和调试支持。
