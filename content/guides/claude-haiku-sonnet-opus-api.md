#### Claude 3 Haiku

Haiku是Claude 3系列中最快速、最实惠的模型。它具备最先进的视觉能力，在行业基准测试中表现出色，是广泛应用于企业应用的多功能解决方案。Haiku模型现已与Sonnet和Opus一同推出，供Claude API和claude.ai上的Claude Pro订阅用户使用。Haiku比同类模型快三倍，大部分工作负载下的处理速度可达每秒21K个令牌（约30页），适用于32K令牌以下的提示。此外，它生成的输出速度快，能够提供响应迅速、引人入胜的聊天体验，同时执行多个小任务。Haiku的定价模型采用1:5的输入输出令牌比例，适用于通常包含较长提示的企业工作负载。企业可以依靠Haiku快速分析大量文件，例如季度报告、合同或法律案例，成本仅为同类性能模型的一半。例如，Claude 3 Haiku可仅用一美元处理和分析400个最高法院案例或2500张图像。此外，Haiku还优先考虑企业级安全性和稳健性，进行严格测试以降低有害输出和模型越狱的可能性，确保尽可能安全。额外的防御层包括持续系统监控、端点加固、安全编码实践、强大的数据加密协议和严格的访问控制，以保护敏感数据。
