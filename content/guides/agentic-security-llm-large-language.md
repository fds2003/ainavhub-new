## Agentic Security

Agentic Security是一个开源的安全扫描工具，旨在对LLM（Large Language Model）进行全面的模糊测试和攻击技术检测。它具有自定义规则集、Agent based攻击等特性，可集成LLM API并进行压力测试。Agentic Security还支持自定义Huggingface数据集和本地CSV数据集。该工具提供了安装和快速启动指南，并支持UI和LLM参数。同时，Agentic Security也提供了关于扩展数据集、添加攻击模板以及动态数据集变化的详细信息。当然，需要注意的是，Agentic Security并不是一个百分之百的安全解决方案，无法保证对所有威胁提供完全保护。
