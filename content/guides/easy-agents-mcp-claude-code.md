## Easy Agents

Easy Agents是一款创新的开源框架，旨在通过自然语言指令和模型上下文协议（MCP）工具调用，替代传统脚本来构建智能自动化代理。该项目核心在于利用先进的大型语言模型（如Claude Code）将高层次的任务目标转化为可执行的自动化工作流，能够智能地编排复杂任务并处理多源数据，实现对安全警报调查、代码安全修复、供应链漏洞管理以及团队周报生成等多样化运营场景的自动化处理。其关键优势在于通过标准化协议简化外部工具集成，并利用可维护的自然语言定义自动化逻辑，极大地降低了团队创建、理解和扩展自动化流程的门槛。
