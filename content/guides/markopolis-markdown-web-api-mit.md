## Markopolis

**Markopolis 简介**

Markopolis 是一个用于处理 Markdown 文件的自托管 web 应用及 API 服务器。用户可以将 Markdown 笔记分享为网站，并通过 API 来互动和操作这些文件。使用者只需安装 Markopolis，并指向存放 Markdown 文件的目录，系统会自动处理后续的工作。Markopolis 是开源的，遵循 MIT 许可协议。其目标是为用户提供一个与 Obsidian Publish 类似的解决方案，但不依赖于 Obsidian 的生态系统。

**使用场景**

1. **个人笔记管理** : 用户可以将 Markdown 笔记管理在本地并通过 Markopolis 分享为网站，适合学生、研究人员或任何需要记录想法的人。
2. **知识共享** : 学者和创作者可以通过 Markopolis 将作品或研究成果以 Markdown 格式发布为在线文档，便于分享和传播知识。
3. **团队协作** : 团队成员可以使用 Markopolis 来整理项目文档、会议记录等，通过 API 进行文档的动态更新和管理。
4. **自定义网页** : 开发者可以利用 Markopolis 的 REST API 构建自定义前端，实现更具个性化的文档展示方式。
5. **Markdown 生态整合** : 对于使用 Obsidian 的用户，Markopolis 提供了兼容的 Markdown 语法，这使得用户可以方便地迁移和整合已有的内容。


Markopolis 的简易部署、对 Obsidian 的兼容性以及丰富的 API 功能，使其成为一个非常灵活和强大的 Markdown 文件管理和发布解决方案。
