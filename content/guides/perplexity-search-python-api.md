## Perplexity Search

Perplexity Search 是一种强大的 Python 工具，采用 Perplexity API 进行技术搜索，旨在提供精准的信息、代码示例和数值数据。这款命令行工具和 Python 库专为开发者、研究人员和技术用户设计，能够快速获取所需的技术文档与信息。

##### 主要特点：

* **互动模式** ：支持多次查询的对话接口，能够保持查询上下文。
* **Markdown 输出** ：可以将对话历史保存在 Markdown 文件中。
* **灵活的模型支持** ：支持不同规模的 LLaMA 模型，可针对具体需求选择。
* **命令行界面** ：易于使用的命令行操作方式。
* **精准的信息查询** ：特别优化用于提取技术信息和代码示例，适合事实和数值数据的检索。


##### 使用场景：

1. **软件开发** ：开发者可以使用该工具快速获取编程语言的性能信息和最佳实践代码。
2. **技术研究** ：研究人员能够查询相关的技术文献、数据统计及详细信息。
3. **教育培训** ：在教学中，能够帮助学生快速了解各种技术概念和解决方案。
4. **故障排除** ：技术支持人员可以利用该工具快速查找解决方案和相关文档，提升问题解决效率。


通过以上功能和使用场景，Perplexity Search 为追求高效和精确的技术查询提供了有力的支持。
