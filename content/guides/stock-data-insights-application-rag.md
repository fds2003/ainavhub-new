## Stock Data Insights Application

Stock Data Insights Application（股票数据洞察应用）是一个利用代理检索增强生成（RAG）工作流，从与特定公司和整体股票市场相关的新闻及金融数据中提取洞察的项目。该应用运用了大型语言模型（LLMs）、ChromaDB作为向量数据库、LangChain及LangChain表达语言（LCEL）和LangGraph，提供全面的分析。

##### 主要功能

* **股票表现可视化** ：展示所选股票的历史表现图表和图形。
* **属性特定数据检索** ：获取与特定股票相关的详细信息。
* **新闻聚合** ：提供与特定股票或公司的综合新闻或主题特定文章。


##### 使用场景

1. **投资分析** ：用户可以查看某支股票的历史表现，获取相关的新闻和财务数据，从而帮助做出投资决策。
2. **市场动态监测** ：实时抓取特定股票的新闻数据，了解其在市场上的动态变化和相关新闻事件，支持投资者及时反应市场情况。
3. **数据研究与可视化** ：对于从事数据分析和研究的用户，通过应用生成的图表和统计数据，深入了解某支股票的价格趋势和表现。
4. **个性化查询** ：用户可以通过自定义查询，获取特定时间段内的股票价格统计，或特定主题下的相关新闻，让使用体验更具灵活性与针对性。


通过这种方式，Stock Data Insights Application为用户提供了一个全面和强大的工具，帮助分析和评估股票市场，实现在复杂信息中的有效决策。
