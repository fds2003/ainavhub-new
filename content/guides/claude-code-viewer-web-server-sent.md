## Claude Code Viewer

Claude Code Viewer 是一个功能全面的、基于网页的 Claude Code 客户端，它提供了一整套交互式功能，用于管理 Claude Code 项目。其核心亮点在于允许用户通过直观的 Web 界面创建新会话、恢复现有对话、实时监控和控制运行中的任务，并能实时同步本地的 Claude Code 项目会话文件。该工具利用 Server-Sent Events (SSE) 实现双向实时通信，自动检测对话文件的更改，提供实时的任务更新和UI自动刷新。此外，它还具备高级会话管理功能，包括项目浏览器、智能会话过滤、多标签界面以及支持语法高亮和工具使用显示的对话展示，全面提升了Claude Code项目的工作效率和用户体验。
