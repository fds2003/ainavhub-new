## Claude Code + Gemini MCP Server

该项目名为 “Claude Code + Gemini MCP Server”，其主要目标是将 Claude Code 与 Google 的 Gemini AI 连接起来，实现强大的 AI 协作。通过该项目，用户可以在 Claude Code 中直接向 Gemini 提问、获取代码审查以及进行头脑风暴。

以下是该项目的主要功能、核心要点和关键特性：

1. **Gemini AI 集成** : 集成了 Google Gemini AI，允许 Claude Code 用户访问 Gemini 的强大功能。
2. **快速启动** : 提供了一键安装脚本，方便用户快速部署和配置。同时也提供了手动安装步骤。
3. **多种工具** : 提供了多种工具，包括 `ask_gemini` (向 Gemini 提问), `gemini_code_review` (获取代码审查), 和 `gemini_brainstorm` (进行头脑风暴)。
4. **MCP Server** : 设置了一个 MCP (Micro Compute Platform) 服务器，桥接 Claude Code 和 Gemini。
5. **全局配置** : 配置为全局可用，可以在任何目录下使用。
6. **问题排查** : 提供了常见问题排查的指南。
7. **API密钥更新** : 方便用户更新 Gemini API 密钥。
