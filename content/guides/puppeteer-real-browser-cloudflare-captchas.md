## Puppeteer Real Browser

##### Puppeteer Real Browser 简介

Puppeteer Real Browser 是一个基于 Puppeteer 的工具库，旨在模拟真实浏览器的行为，以避免在使用自动化脚本时被网站检测为机器人。它可以有效绕过像 Cloudflare 这样的安全服务，并能够自动处理验证码（如 Captchas），从而提升自动化任务的成功率。

此库主要通过针对浏览器的最小修改，保持其自然状态，确保最佳的运行结果，尤其是在需要复杂的用户交互时（例如填写表单、点击按钮等）。

##### 使用场景

Puppeteer Real Browser 适用于多种场景，包括但不限于：

1. **爬虫与数据抓取** ：当需要从网站提取数据时，特别是那些具有较高安全性的站点，Puppeteer Real Browser 可帮助用户避免被网站的检测机制阻挡。

2. **自动化测试** ：在进行网站的自动化测试时，确保测试行为更接近真实用户的操作，有助于提高测试的准确性和有效性。

3. **模拟用户行为** ：在需要模拟复杂的用户交互（如登录、表单填写、在线购物等）场景中，Puppeteer Real Browser 提供了更可靠的解决方案。

4. **绕过 CAPTCHA 验证** ：该库能通过自动点击和处理验证码，帮助用户在执行自动化任务时不被 CAPTCHAs 拦截。
