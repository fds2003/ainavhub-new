## AI Agent Service Toolkit

##### AI Agent Service Toolkit 简介

AI Agent Service Toolkit 是一个全面的工具包，旨在构建和运行基于 AI 的代理服务。它结合了 LangGraph、FastAPI 和 Streamlit，提供了一整套工具，使开发者能够轻松创建并运行自己的 AI 代理。该工具包的特点包括定制化的 LangGraph 代理、FastAPI 服务用于提供接口、交互式客户端和用户友好的 Streamlit 聊天界面，旨在简化基于 LangGraph 项目的构建及使用。

###### 主要特点：

1. **LangGraph 代理** ：定制的代理，利用 LangGraph 框架构建。
2. **FastAPI 服务** ：提供流式和非流式的 API 接口。
3. **高级流式处理** ：支持基于 token 和消息的流式处理。
4. **内容审核** ：实现 LlamaGuard 用于内容审核。
5. **Streamlit 接口** ：简洁的聊天界面，便于用户与代理互动。
6. **多代理支持** ：可以在服务中运行多个代理，并通过 URL 路径调用。
7. **异步设计** ：利用 async/await 处理并发请求。
8. **反馈机制** ：集成 LangSmith 的基于星级的反馈系统。
9. **动态元数据** ：提供服务和可用代理、模型的动态配置元数据。
10. **Docker 支持** ：便于开发和部署的 Docker 配置。


##### 使用场景

AI Agent Service Toolkit 可以应用于多种场景，包括但不限于：

1. **聊天机器人** ：构建智能交互的聊天机器人，为用户提供支持和信息。
2. **虚拟助手** ：开发可以处理任务和回答问题的个人虚拟助手。
3. **实时数据分析** ：创建能够实时分析数据并提供反馈的代理，以支持决策。
4. **内容生成** ：利用代理生成高质量的写作、创意作品或商业内容。
5. **教育应用** ：构建可以执行特定任务或提供学习支持的教育代理。
6. **多代理协作** ：实现多个不同功能的代理之间的协作，实现复杂任务的处理。


通过提供一个易于使用和自定义的框架，AI Agent Service Toolkit 确保了开发者能够高效地创建所需的代理服务，适用于各种行业和应用场景。
