## SuperMemory

## SuperMemory 简介

SuperMemory 是一个帮助用户创建“第二大脑”的工具，相当于一个适用于书签和保存内容的 ChatGPT。它允许用户导入推文或保存网站和内容，通过 Chrome 扩展实现。

##### 主要功能

* **创意捕捉** : 轻松捕捉和保存创意。
* **书签管理** : 导入、组织和重新找到书签。
* **联系人管理** : 存储和管理你认识的人的信息。
* **推特书签导入** : 使用和管理已保存的推文。
* **强大的搜索功能** : 快速找到任何已保存的信息。
* **与知识库对话** : 互动特定的知识库。
* **知识画布** : 在二维画布上可视化组织信息。
* **写作助手** : 使用带有 AI 协助的 Markdown 编辑器进行内容创作。
* **隐私保护** : 确保数据安全和隐私。
* **自我托管** : 开源且易于本地部署。
* **集成** : 兼容 Telegram、Twitter 等，更多集成将会推出。


##### 使用场景

1. **个人知识管理** ：用于存储和管理个人在互联网上找到的重要信息和灵感。
2. **社交媒体内容管理** ：帮助用户保存和管理在推特上的重要推文。
3. **信息快速检索** ：通过强大的搜索功能，快速找到已保存的信息。
4. **内容创作助力** ：利用 AI 和 Markdown 编辑器为写作和内容创作提供帮助。


##### 如何使用

* 注册并登录 [supermemory.ai](<https://supermemory.ai>)。
* 下载 Chrome 扩展并安装。
* 可通过页面右下角的图标保存当前页面。


##### 推特书签导入步骤

1. 登录 SuperMemory 并安装 Chrome 扩展。
2. 打开推特，点击保存图标。
3. 等待 10-20 秒，扩展会同步所有推特书签到 SuperMemory。
4. 完成，所有推特书签都已导入到你的第二大脑。


SuperMemory 的核心技术栈包括 React、Next.js、Tailwind CSS、Cloudflare 等，开源并允许社区贡献。详细的安装和使用指南请参阅 [项目的 GitHub 页面](<https://github.com/Dhravya/Supermemory>)。
