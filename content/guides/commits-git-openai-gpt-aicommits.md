## AI Commits

AI Commits 是一个命令行工具，旨在自动生成 Git 提交消息，用户不再需要手动编写提交信息。它利用 OpenAI 的 GPT 模型来分析代码变更，并生成相关的提交消息，从而大幅提高开发效率。

##### 使用场景

1. **自动生成提交消息** ：开发者在进行代码变更时，只需使用 `aicommits` 命令，工具会自动生成与当前更改相符的提交消息，节省编写时间。

2. **生成多个候选消息** ：用户可以请求生成多条提交消息，从中选择最合适的一条，这在决定如何描述提交时特别有用。

3. **支持规范化提交** ：可以按照“常规提交”规范生成消息，适用于对代码提交格式有严格要求的项目。

4. **与 Git 钩子集成** ：可以将 AI Commits 集成到 Git 的提交流程中，即在每次提交时自动生成消息，同时仍然允许用户在必要时手动输入消息。

5. **多种配置选项** ：支持配置 API 密钥、生成消息数量、区域设置等，满足不同开发者的需求。


通过以上功能，AI Commits 提供了一种高效、灵活的方式来管理版本控制系统中的提交消息，提高团队的协作效率和代码质量。
