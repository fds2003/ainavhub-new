## Xiaozhi Client

Xiaozhi Client 是一个多功能的AI客户端，主要用于MCP（微服务通信协议）集成。其核心功能包括作为官方小智AI服务的客户端，并能以标准MCP Server的形式集成到Cursor、Cherry Studio等第三方客户端，从而实现MCP配置的集中管理与多设备共享。它支持配置和聚合多个小智AI接入点，动态控制MCP工具的可见性，并能无缝对接本地部署的开源服务端或远程ModelScope托管的MCP服务。项目提供了强大的命令行工具集和现代化的Web UI界面，极大地简化了MCP服务的可视化配置与管理，避免了手动编辑JSON文件的繁琐。此外，它严格遵循JSON-RPC 2.0通信规范，明确定义了请求、通知和响应格式，便于开发者构建兼容的自建MCP服务端，并支持stdio和HTTP Server两种集成模式，提供了灵活的部署选择。
