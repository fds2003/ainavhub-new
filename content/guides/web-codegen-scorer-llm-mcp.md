## Web Codegen Scorer

Web Codegen Scorer是一个专注于评估大型语言模型（LLM）生成Web代码质量的工具，旨在帮助用户做出基于证据的AI生成代码决策。其核心功能包括灵活配置不同模型、框架和工具的评估环境，支持系统指令和MCP服务器集成。该工具提供了一系列内置检查，涵盖代码构建成功、运行时错误、可访问性、安全性、LLM评分以及编码最佳实践，并能自动尝试修复检测到的问题。通过直观的报告界面，用户可以轻松查看和比较评估结果，从而高效迭代系统提示、对比不同模型产出的代码质量，并持续监控代码生成质量随时间的变化。该项目由Google的Angular团队构建，但设计上与特定框架或模型无关，强调对Web代码的独特关注和对既定代码质量标准的依赖，以提供一致且可重复的评估平台。
