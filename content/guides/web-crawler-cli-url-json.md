## Web Crawler

Web Crawler是一个精简的实时命令行界面（CLI）网络爬虫，旨在帮助用户快速搜索互联网。它能够根据用户输入的查询，即时返回包含标题、URL和发布日期的JSON格式搜索结果，并按时间新旧排序。用户可以通过CLI交互式地输入查询，获取结果后可继续输入新的查询或通过简单命令退出。
