## rallies-cli

`rallies-cli` 是一个由 Rallies 团队开发的 AI 驱动的命令行投资研究工具，其核心在于将人工智能的对话能力与实时金融数据相结合，从而为交易者和投资者提供类似 ChatGPT 但拥有最新市场信息的分析平台。与依赖过时或非实时网络数据的通用 AI 工具不同，`rallies-cli` 能够处理复杂的实时查询，例如分析当日期权流向或查找当日股票模式，并提供准确且当前的市场分析。其主要功能包括执行基础查询、进行深度高级分析、执行技术分析以及跟踪实时新闻和市场事件，帮助用户获取即时、专业的金融洞察。该工具需要 OpenAI API 密钥，并可通过 Rallies.ai API 密钥获得更高的使用限制，但需注意其仅供教育和研究目的，不提供任何投资建议。
