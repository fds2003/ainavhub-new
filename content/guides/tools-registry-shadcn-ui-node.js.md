## AI Tools Registry

AI Tools Registry项目是一个基于`shadcn/ui`构建的可安装AI工具注册表，它专注于为Node.js环境提供一套利用Vercel AI SDK工具调用API开发的实用AI工具，如天气、计算器、翻译、新闻等。其核心在于允许用户通过`shadcn CLI`便捷地将这些工具及其附带的React渲染组件（例如天气卡片、新闻列表）集成到自己的项目中，并支持对工具的独立或批量安装，同时提供Web搜索集成以及认证功能的示例。
