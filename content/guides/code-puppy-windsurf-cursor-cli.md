## Code Puppy

Code Puppy是一个AI驱动的代码生成代理，旨在理解编程任务，生成高质量的代码，并解释其推理过程，类似于Windsurf和Cursor等工具。

主要功能和特点：

1. **多语言支持：**能够生成多种编程语言的代码。
2. **交互式CLI：**提供一个命令行界面用于交互式使用。
3. **详细的解释：**提供对生成的代码的深入理解，以便理解其逻辑和结构。
4. **可定制的模型配置：**允许用户指定自定义模型，例如使用Azure OpenAI或者具有特定端点和头信息的定制模型。
5. **支持Puppy Rules：** 可以定义和强制执行代码标准和风格，例如指定应用程序的设计准则，如深色模式主题和青色强调色。
