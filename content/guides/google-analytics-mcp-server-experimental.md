## Google Analytics MCP Server

“Google Analytics MCP Server (Experimental)”项目是一个本地模型上下文协议（MCP）服务器，旨在通过整合Google Analytics Admin API和Data API，为大型语言模型（LLMs），特别是Gemini CLI/Code Assist，提供一套工具，实现对Google Analytics数据的深度交互与智能分析。其核心功能在于支持LLM通过自然语言查询来检索账户和属性详情、执行核心报告和实时报告，并获取可用的维度与指标，从而简化了用户与复杂Google Analytics数据之间的交互，使数据分析更加直观和高效。
