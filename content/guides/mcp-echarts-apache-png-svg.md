## MCP ECharts

MCP ECharts 是一个强大且安全的工具，它利用 AI MCP 动态生成 Apache ECharts 图表，旨在提供灵活的数据可视化和分析能力。该项目全面支持 ECharts 的所有功能和语法，并能将图表导出为 PNG、SVG 或原始配置，同时进行语法验证以确保输出的准确性。其核心优势在于轻量级设计、零依赖以及完全本地生成的安全性。此外，它还支持与 MinIO 等对象存储服务集成，可将生成的图表存储为 URL 以提升性能和分享便利性，并在未配置时无缝切换回 Base64 输出。项目提供了多种传输协议支持，方便在桌面应用或作为独立服务中灵活部署和使用。
