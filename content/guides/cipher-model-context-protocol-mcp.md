## Cipher

Cipher是一个专为编码智能体设计的开源记忆层框架，核心功能在于通过深度集成Model Context Protocol（MCP），为Cursor、Windsurf、Claude Code等主流IDE及AI编码智能体提供持久化、跨平台无缝共享的记忆能力。其关键特性包括独特的双层记忆系统，能够捕捉编程概念、业务逻辑以及AI生成代码时的推理过程，实现记忆的自动生成和实时优化，并支持团队实时共享记忆工作空间。该项目具备多种操作模式（如CLI、API、MCP服务器），兼容OpenAI、Anthropic、OpenRouter、Ollama等多个大型语言模型提供商，同时提供会话管理和知识图谱等高级功能。
