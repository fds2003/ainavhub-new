## Claude Code but with OpenAI Models

**总结性介绍：**

`Claude Code but with OpenAI Models` 是一个代理服务器，它允许你使用 Anthropic 的 Claude Code 工具，但实际上底层使用的是 OpenAI 的模型，例如 GPT-4o 和 gpt-4o-mini。简单来说，它就像一个翻译器，将 Claude Code 发出的请求转换为 OpenAI 模型可以理解的格式，并将 OpenAI 的回复转换回 Claude Code 可以理解的格式。这样，你就可以利用 Claude Code 的界面和功能，同时享受到 OpenAI 模型的强大能力。

**使用场景：**

* **利用 OpenAI 的先进模型：** 如果你喜欢 Claude Code 的使用方式，但更希望使用像 GPT-4o 这样的 OpenAI 最新、最强大的模型，这个代理服务器能帮你实现。
* **模型选择的灵活性：** 你可以根据需要，将 Claude 的不同模型 (例如 haiku 和 sonnet) 映射到不同的 OpenAI 模型 (例如 gpt-4o 和 gpt-4o-mini)。
* **实验和比较：** 它可以让你比较 Claude Code 与不同 OpenAI 模型组合的效果，选择最适合你需求的配置。
* **绕过限制：** 在某些情况下，你可能无法直接访问或使用 Anthropic 的 Claude 模型，但可以通过这个代理服务器，利用 OpenAI 模型来实现类似的功能。
* **二次开发：** 开发者可以在这个基础上进行二次开发，例如添加更高级的请求转换或响应处理逻辑。


总而言之，`Claude Code but with OpenAI Models` 提供了一种桥梁，连接了 Claude Code 的使用体验和 OpenAI 模型的强大能力，让用户拥有更灵活、更强大的 AI 工具。
