## POC Python Realtime API o1 assistant

##### POC Python Realtime API o1助手简介

POC Python Realtime API o1助手是一个概念验证项目，旨在利用OpenAI的实时API，实现工具链的调用、o1-preview和o1-mini的集成、结构化输出的响应处理，从而展望未来的智能助手工程。该项目包含许多核心思想，旨在打造个人化的实时AI助手与AI代理。虽然目前的代码版本（v0.3）存在一些bug，但提供了丰富的功能和使用场景。

该助手通过音频输入/输出、多种工具集成和记忆管理，实现与用户的互动，支持个性化设置。用户可以通过修改配置文件来定制助手的行为，包括浏览器链接、助理名称、SQL方言等。

##### 使用场景

1. **语音命令操作** ：用户能够通过自然语言与助手互动。例如，可以询问当前时间、生成随机数、创建或更新CSV文件、执行SQL查询等。

2. **CLI文本提示** ：用户也可以通过命令行输入多个提示，使用’|‘分隔指令串联调用，为助手发出多项任务指令。

3. **智能助手** ：该助手可以用于构建和管理日常工作任务，帮助用户生成文件、执行代码、抓取网页信息等。

4. **数据库操作** ：助手支持多种SQL方言，能够进行数据库查询和数据管理，适用于需要数据库处理的应用场景。

5. **动态创建图表和架构设计** ：用户可以请求助手生成图表，规划系统架构等，帮助视觉化技术设计。

6. **持续记忆功能** ：助手具有记忆管理功能，可以在不同会话间持续存储信息，使用户能够随时访问和更新之前的状态。


通过灵活的命令与个性化的配置，POC Python Realtime API o1助手可以被广泛应用于日常工作自动化、数据处理、编程辅助及智能助理等多个领域。
