## Cursor Chat Browser

**Cursor Chat Browser简介：**

Cursor Chat Browser是一款用于浏览和管理Cursor编辑器中AI聊天功能的聊天历史记录的Web应用程序。用户可以查看、搜索并以多种格式导出其AI对话记录。这款应用程序具有简洁的界面和强大的功能，旨在提升用户对聊天记录的管理和使用体验。

**主要特性：**

* 支持浏览和搜索所有工作区的聊天历史
* 可查看AI聊天记录和Composer日志
* 按工作区组织聊天记录
* 提供全文搜索功能，并可对聊天和Composer日志进行过滤
* 响应式设计，支持深色模式和浅色模式
* 可将聊天记录导出为Markdown文件、HTML文档（带语法高亮）和PDF文件
* 支持语法高亮的代码块
* 可书签聊天URL，方便快速访问
* 自动检测工作区路径，简化用户操作


**使用场景：**

1. **信息检索** ：用户可以快速查找特定的聊天记录或信息，通过强大的搜索功能和过滤器，轻松找到所需内容。

2. **记录管理** ：在多个工作区内管理聊天记录，适用于不同项目或讨论主题，方便用户有条理地存储和查阅信息。

3. **知识分享** ：用户可以将对话导出为Markdown、HTML或PDF格式，便于分享给其他团队成员或在文档中引用。

4. **界面个性化** ：根据个人偏好选择深色或浅色模式，提升用户的使用体验。

5. **开发者使用** ：对于开发者来说，可以在应用中直接查看、测试和调试AI聊天记录，有助于改进和优化AI模型的表现。


总之，Cursor Chat Browser为希望有效管理和利用AI聊天记录的用户提供了一个非常有价值的工具。
