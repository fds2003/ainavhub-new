## Claude Code Spec-Driven Development

Claude Code Spec-Driven Development是一个基于Claude Code斜杠命令和`CLAUDE.md`的项目，旨在系统化并高效地实施规格驱动开发（Spec-Driven Development, SDD）。它通过提供一系列集成命令，自动化了开发流程中的关键文档生成，包括项目引导文档（如产品、技术、架构规范）、功能需求、技术设计和具体实施任务。该项目核心在于强制执行严格的多阶段人工评审与批准工作流，确保每个开发阶段的质量与一致性，同时利用自动化机制实现任务进度追踪、规格合规性检查，并在整个开发周期中保持项目上下文和防止方向性偏差，从而提升开发效率和产品质量。
