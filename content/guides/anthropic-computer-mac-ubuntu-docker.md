## Anthropic Computer Use

**Anthropic Computer Use 简介**

Anthropic Computer Use（适用于Mac）是一个测试版功能，旨在通过运行Ubuntu的Docker镜像来控制Mac系统。该功能让用户能够直接使用本地macOS命令和工具，对系统进行控制。这一版本的Anthropic允许用户在macOS上本地运行，提供完善的系统交互。

**功能特点**

* 直接与macOS GUI交互，无需Docker环境。
* 支持使用macOS本地命令进行屏幕截图。
* 通过cliclick实现对键盘和鼠标的控制。
* 支持多个大型语言模型提供商（如Anthropic, Bedrock, Vertex）。
* 基于Streamlit的用户界面。
* 自动调整屏幕分辨率。
* 文件系统交互和编辑功能。


**使用场景** Anthropic Computer Use 可用于多种场景，包括：

1. **自动化任务** ：可以通过自然语言接口编写代码来自动执行日常计算机任务，例如文件管理和数据处理。
2. **图形界面控制** ：直接与macOS的图形界面进行交互，适合需要图形化响应的应用场景。
3. **机器学习实验** ：在研究和开发AI模型时，利用多种支持的LLM来测试和优化系统性能。
4. **技术演示和教育** ：用于教授编程和系统控制的概念，通过交互式界面演示复杂的操作。


需要注意的是，Anthropic代理可以控制Mac上的所有内容，因此用户在使用时需谨慎。此外，Anthropic的Claude 3.5 Sonnet模型在保护用户安全方面具有一定的限制，不会执行危害安全的操作，如购买物品或下载非法内容。
