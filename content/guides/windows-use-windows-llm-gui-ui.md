## Windows-Use

Windows-Use 是一个先进的Windows自动化代理，其核心功能在于让任意大型语言模型（LLM）能直接通过图形用户界面（GUI）层控制Windows操作系统，执行打开应用、点击、输入、运行命令及捕获UI状态等任务，显著区别于传统依赖计算机视觉模型的方法，从而实现了AI Agent与Windows的无缝桥接。尽管它提供高效的自动化能力，但鉴于其直接操作系统的特性，项目方强调使用时需注意潜在风险，并建议在沙盒环境中运行以防范意外行为。
