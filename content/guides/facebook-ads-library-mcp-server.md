## Facebook Ads Library MCP Server

好的，我来总结一下这个名为 “Facebook Ads Library MCP Server” 的 GitHub 项目的主要功能、核心要点和关键特性。

这个项目是一个用于 Facebook 广告库的模型上下文协议 (MCP) 服务器。它允许用户：

1. **搜索并分析 Facebook 广告库** : 能够搜索任何公司或品牌在 Facebook 上的公开广告库，查看他们当前正在投放的广告并进行分析。
2. **广告分析** : 分析广告的图像/文本内容，获取视频链接，比较不同公司的广告策略，并深入了解其广告活动的效果。
3. **与 Claude 集成** : 该服务器可以与 Anthropic Claude (Desktop 或 Cursor) 集成，用户可以直接通过 Claude 使用自然语言查询来分析 Facebook 广告数据。
4. **Scrape Creators API** : 该项目依赖于 Scrape Creators API 来获取 Facebook 广告数据，用户需要注册并获取 API 密钥。
5. **MCP 工具** : 提供 `get_meta_platform_id` 和 `get_meta_ads` 等 MCP 工具，用于获取 Meta 平台 ID 和检索特定页面的广告。
6. **无需代码** : 提供了一个无需编写代码的版本，方便用户使用。


总而言之，这个项目的主要目标是为用户提供一个方便易用的接口，以便通过 Claude 等工具分析 Facebook 广告数据，帮助他们了解竞争对手的广告策略，并优化自己的广告活动。
