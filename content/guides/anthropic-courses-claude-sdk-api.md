## Anthropic courses

Anthropic课程是一个针对用户学习和掌握Claude SDK及相关技术的教育资源库，目前包含四个课程。建议按照以下顺序进行学习：

1. **Anthropic API基础课程** ：该课程介绍了使用Claude SDK的基础知识，如获取API密钥、调整模型参数、撰写多模态提示、流式响应等。
2. **提示工程互动教程** ：这是一个全面的逐步指导，涵盖了关键的提示技术。
3. **现实世界提示课程** ：该课程教你如何在复杂的实际场景中应用提示技术。
4. **工具使用课程** ：学习如何在工作流程中成功实现工具的使用。


##### 使用场景

Anthropic课程适用于希望深入理解和应用人工智能模型的开发者和研究者。通过这些课程，用户能够学习到API的基本操作、提示设计的技巧、在实际应用中的提示方法以及如何有效整合工具来提升工作的效率。这些知识和技巧能够帮助用户在实际项目中更好地利用Claude SDK，从而实现更高效、更智能的工作流。
