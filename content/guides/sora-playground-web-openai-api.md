## Sora 2 Playground

Sora 2 Playground是一个基于Web的交互平台，专为OpenAI的Sora 2模型设计，用户能够通过文本提示生成全新视频，或利用视频混音模式对现有视频进行定向修改。该项目提供了对Sora 2 API所有关键参数的全面UI控制，支持多视频并发生成队列，并集成详细的历史记录与成本追踪功能，用户可以查看每次生成请求的参数、美元成本细目、实时进度，并方便地管理和删除视频。此外，它具备视频输出面板、快速发送至混音功能及视频下载能力，并在存储上灵活支持服务器文件系统或浏览器IndexedDB模式，尤其适合Vercel等无服务器环境部署，但使用前需具备OpenAI Sora 2 API的访问权限。
