## QueryWeaver

QueryWeaver是一款开源的Text2SQL工具，它通过图驱动的模式理解将自然语言问题转化为SQL查询并返回结果。该项目的核心亮点在于其强大的自然语言到SQL转换能力，通过REST API管理数据库图谱和执行查询，并可选地支持模型上下文协议（MCP），同时灵活集成了Azure OpenAI或直接OpenAI等AI/LLM模型以增强分析能力，并通过OAuth或API令牌提供安全的用户认证。
