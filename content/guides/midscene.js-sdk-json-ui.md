## Midscene.js

Midscene.js 是一个由 AI 驱动的自动化 SDK，能够通过自然语言控制网页、执行断言，并以 JSON 格式提取数据。其主要功能包括：

* **自然语言交互** ：用户只需描述操作步骤，Midscene.js 即可自动规划和控制界面。
* **理解 UI 并以 JSON 格式回复** ：提供关于所需数据格式的提示，系统会返回预期的 JSON 格式数据。
* **直观断言** ：基于 AI 理解的自然语言断言。
* **开箱即用的多模态大模型** ：支持公共的多模态大模型如 GPT-4，无需进行自定义训练。
* **可视化** ：通过可视化工具，用户可以轻松理解和调试整个过程。
* **全新体验** ：享受前所未有的自动化开发新体验。


##### 使用场景

Midscene.js 适用于以下场景：

* **网页自动化测试** ：通过自然语言描述测试步骤，而非传统的代码脚本编写。
* **数据提取和报告生成** ：从网页中提取数据并生成格式化的 JSON 数据报告。
* **UI 监控和自动化操作** ：自动监控网页并执行相应的操作，如定时刷新页面、自动提交表单等。
* **增强的用户体验** ：通过简单描述实现复杂的界面交互和操作。
