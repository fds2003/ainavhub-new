## Claude Computer Use Demo for MacOS

Claude Computer Use Demo for MacOS 是一个展示 Anthropic 计算机使用能力的 Python 脚本，特意优化为在 MacOS 系统上运行，无需 Docker 容器。该脚本允许 Claude 3.5 Sonnet 通过模拟鼠标和键盘操作以及运行 bash 命令来执行任务。

##### 使用场景

1. **自动化任务** ：用户可以使用该脚本来自动打开应用程序（例如 Safari），进行信息搜索或执行其他常见任务，从而提高工作效率。
2. **代码和命令行操作** ：该工具能够模拟用户在终端中的输入，为开发者提供了一种方便的方式来运行命令和脚本。
3. **实验和演示** ：对于研究人员和开发者而言，该脚本可以用于展示 Claude 的能力，进行技术演示和实验。
4. **界面互动** ：用户也可以让 Claude 通过鼠标和键盘与 MacOS 界面进行互动，执行一些复杂的用户行为。


##### 安装与设置

用户需要克隆代码库，创建虚拟环境并安装所需依赖，设置 API 密钥，并授权可访问性权限。启动脚本后，用户可以通过命令行提供指令来执行操作。

##### 注意事项

使用此脚本时需谨慎，因为它允许 Claude 控制计算机的鼠标和键盘，存在一定的安全风险。用户须对运行结果承担全部责任和风险。
