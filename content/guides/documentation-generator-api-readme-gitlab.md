## AI Documentation Generator

AI Documentation Generator 是一款由人工智能驱动的代码文档生成工具，其核心在于采用多智能体架构，能够对代码库进行深入分析，包括代码结构、数据流、依赖、请求流和API等方面。该项目的主要功能是自动生成包含可配置章节的README文档，并支持GitLab集成以自动化分析和合并请求创建。它具备多任务并发处理能力以优化性能，提供灵活的YAML配置与命令行覆盖选项，并兼容多种OpenAI API标准的大语言模型，同时内置OpenTelemetry和Langfuse实现可观测性。
