## Browser Echo

Browser Echo 是一个将浏览器 `console.*` 日志实时流式传输到开发终端的工具，并支持可选的文件日志记录，旨在简化开发者和 AI 编码助手在开发过程中读取客户端日志的流程。其核心亮点在于深度支持 AI 编码助手（如 Cursor AI, Claude Code 等），使日志分析更加高效。该项目兼容主流前端框架（如 React, Vue, Next.js, Nuxt, Vite），且设计上确保对生产环境无任何影响，仅在开发模式下生效。它提供了一系列实用功能，包括自动封装 `console` 方法、支持批量日志发送、显示日志源文件信息与堆栈追踪、终端彩色输出以及高度可定制的日志级别、显示模式和批处理设置，同时保持轻量化，避免框架间不必要的依赖臃肿。
