## Ai-Review

AI-Review 项目专注于利用人工智能技术，优化论文撰写及投稿准备工作，旨在帮助研究人员高效地识别手稿的优点与不足，并获取具体的改进建议。它提供了一个便捷的Web界面，用户可以直接通过“快速尝试”功能体验其强大能力，无需自行配置API，同时支持接入Deepseek API。该项目的一大亮点是具备数学符号和公式的强制性漏洞检查功能，并通过精心优化的Prompt，能够生成结构清晰、内容详尽的评审意见，包括以二级列表形式呈现的优点、缺点和改进建议，从而有效提升论文的质量和接收率。
