## AutoDocs

AutoDocs 是由 Sita 推出的一个强大的自动化文档生成工具，旨在深度遍历代码库、解析抽象语法树（AST）并构建代码依赖图，进而自动生成准确、高质量且具备依赖感知的仓库级文档和摘要。该项目提供了一个 FastAPI 后端用于代码摄入与搜索，以及一个 Next.js Web UI 界面用于文档的聊天和探索，并内置了一个 MCP 服务器，使编码代理能够通过 HTTP 对代码进行深度查询，以回答仓库范围内的相关问题。它支持多种主流编程语言，并计划持续扩展对更多语言及多语言仓库的支持。
