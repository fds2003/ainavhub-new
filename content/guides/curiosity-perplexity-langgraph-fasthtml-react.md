## curiosity

**Curiosity 概述**

Curiosity 是一个用于探索和实验的项目，旨在搭建一个类似 Perplexity 的用户体验。该项目结合了多种技术框架，如 LangGraph 和 FastHTML，并使用 ReAct Agent 进行文本生成，同时通过 Tavily 搜索功能增强生成的内容。尽管项目的核心是构建一个功能性的聊天机器人，但开发过程中同样注重用户界面的视觉效果和交互体验。

**使用场景**

Curiosity 可以应用于多个场景，具体包括：

1. **聊天机器人开发** ：利用不同的语言模型（如 gpt-4o-mini、llama3-groq 等）创建智能对话系统。
2. **信息检索** ：通过 Tavily 搜索集成，增强用户获取信息的效率和准确性。
3. **前端开发实验** ：开发者可以使用 FastHTML 构建快速响应的用户界面，体验现代 web 开发框架的特性。
4. **技术学习和探索** ：对使用最新 AI 技术和工具感兴趣的开发者可以借助这个项目来学习和实验，如 API 的集成和性能调优。
5. **个性化项目开发** ：Curiosity 的灵活性允许开发者根据具体需求进行调整或拓展，可以作为各类 AI 应用的基础。


整体而言，Curiosity 是一个通过引入多种现代技术栈来增强用户体验和功能的实验性项目，适合对 AI 和前端开发感兴趣的技术人员。
