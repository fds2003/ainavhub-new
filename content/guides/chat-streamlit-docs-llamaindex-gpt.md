## Chat with the Streamlit docs

Chat with the Streamlit docs 是一个基于Streamlit开发的聊天机器人应用，它利用LlamaIndex技术和GPT 3.5模型，通过索引和调用Streamlit官方文档内容来回答用户关于Streamlit的查询。这个功能在你需要快速获取Streamlit库相关问题的解答时非常有用，尤其是当你在开发中遇到疑惑，或者想了解Streamlit特定功能的详细信息时。

##### 应用概览

* 应用允许用户通过Streamlit的`st.chat_input`输入查询，然后通过`st.chat_message`展示用户的查询和模型的回答。
* 它使用LlamaIndex来加载和索引数据，创建一个聊天引擎，这个引擎会从索引的数据中检索上下文，从而对每个用户查询提供回答。


##### 使用情况

* 当你在使用Streamlit开发应用，遇到了关于库使用的具体问题时，比如如何实现某个特定的功能，或者对一些API的参数配置有疑问。
* 当你想要快速获取Streamlit官方文档中的信息，而不想花费时间在文档中逐页搜索。
* 当你进行Streamlit学习和研究，需要即时帮助来更深入理解某些概念或功能。


##### 如何使用

1. 通过访问Streamlit应用链接，打开Chat with the Streamlit docs应用。
2. 如果需要，按照指导获取OpenAI API密钥，并在应用中进行配置，以便让聊天机器人能够运行。
3. 在聊天输入框中输入你对Streamlit有关的问题。
4. 提交问题后，等待系统处理并返回基于Streamlit文档内容的答案。
