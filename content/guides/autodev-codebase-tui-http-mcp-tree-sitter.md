## autodev-codebase

`autodev-codebase` 是一个平台无关的代码分析库，专注于通过智能代码索引和向量化语义搜索来提升开发效率。该项目提供了一个交互式命令行界面（TUI）供直接使用，更关键的是，它支持基于 HTTP 的模型上下文协议（MCP）服务器，使得其强大的语义搜索功能和通过 Tree-sitter 实现的高级代码解析能力能够无缝集成到各种 IDE 中。用户可以通过分层配置系统高度定制化其行为，包括选择不同的嵌入模型（如通过 Ollama 或兼容 OpenAI 的服务）和指定 Qdrant 作为向量数据库，从而提供一个灵活且高效的代码库导航和理解解决方案。
