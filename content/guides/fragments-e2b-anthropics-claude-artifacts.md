## Fragments by E2B

**Fragments by E2B 介绍**

Fragments by E2B 是一个开源项目，基于 Anthropics 的 Claude Artifacts、Vercel v0 和 GPT Engineer 开发。该项目利用 E2B SDK 来安全地执行由人工智能生成的代码，支持多种编程语言和技术栈。

**主要特点：**

* 基于 Next.js 14，采用App Router和Server Actions，使用 shadcn/ui 和 TailwindCSS，集成 Vercel AI SDK。
* 在用户界面中支持流式数据传输。
* 可以从 npm 和 pip 安装和使用任意包。
* 支持多种技术栈，包括 Python 解释器、Next.js、Vue.js、Streamlit 和 Gradio。
* 支持多个大型语言模型（LLM）提供商，如 OpenAI、Anthropic、Google AI 等。


**使用场景：** Fragments by E2B 可用于以下场景：

1. **教育与学习** ：学生可以通过这个平台学习编程、数据科学和机器学习等技术。
2. **开发与原型设计** ：开发者可以利用其支持的多种技术栈快速构建和测试应用原型。
3. **数据分析** ：可用于执行数据分析相关的代码片段，支持多种数据可视化和分析库。
4. **API 集成** ：利用可自定义的 LLM 提供商来完成不同的自然语言处理任务，便于集成各种API。
5. **开源贡献** ：作为一个开源项目，社区可以对此进行贡献，如修复bug或增加新特性。


总而言之，Fragments by E2B 是一个强大的工具，适用于开发、学习和数据分析等多个领域。
