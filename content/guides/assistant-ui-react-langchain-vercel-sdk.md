## assistant-ui

**assistant-ui** 是一套用于AI聊天的React组件，支持多种集成，包括Langchain、Vercel AI SDK、TailwindCSS、shadcn-ui、react-markdown、react-syntax-highlighter、React Hook Form等。

该工具箱支持多种模型提供商（如OpenAI、Anthropic、Mistral、Perplexity、AWS Bedrock、Azure、Google Gemini、Hugging Face等），并且可以轻松集成自定义API。

##### 使用场景

1. **客户支持** ：可以创建智能聊天助手，帮助用户解决常见问题或提供服务。
2. **教育** ：用于构建学习助手，帮助学生提问和获取信息。
3. **内容创作** ：可以协助用户撰写文章、生成创意内容或进行内容优化。
4. **开发工具** ：为开发者创建调试助手或代码提示工具，提升编码效率。


##### 快速开始

1. 使用 `assistant-ui` 创建新项目，命令如下：

npx assistant-ui@latest create my-app
cd my-app


2. 更新 `.env` 文件，输入您的OpenAI API密钥。
3. 运行应用程序：

npm run dev


通过以上步骤，您将快速启动一个功能强大的AI聊天应用，满足多种业务需求。
