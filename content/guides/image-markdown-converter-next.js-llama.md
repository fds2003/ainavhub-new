## Image to Markdown Converter

##### Image to Markdown Converter 简介

Image to Markdown Converter 是一个基于 Next.js 的应用程序，能够将图像转换为 Markdown 格式，使用了 Llama 3.2 Vision 模型进行图像处理。用户可以通过上传图片，实时预览生成的 Markdown 内容，并方便地复制到剪贴板。此外，该应用程序支持响应式设计、拖放上传以及夜间模式。

##### 使用场景

1. **内容创作者** ：博客作者和在线写手可以快速将图像中的文本内容提取并转换为 Markdown，方便添加到他们的文章中。
2. **开发者文档** ：技术文档编写者可以方便地从截图中提取信息，简化文档创建过程。
3. **教育用途** ：教师和学生可以从课件或书籍中的图像中提取和转换文本，用于学习和复习资料。
4. **社交媒体** ：用户可以将社交媒体上的图片内容快速转为 Markdown ，方便在不同平台之间分享和引用。


##### 特性

* 支持图像上传并转换为 Markdown 格式
* 实时 Markdown 预览
* 拖放上传功能
* 复制到剪贴板
* 响应式设计
* 暗黑模式支持


##### 技术栈

* **Next.js** ：用于构建页面和处理路由
* **React** ：构建用户界面
* **Tailwind CSS** ：用于快速样式定制
* **Together AI** ：提供 Llama 3.2 Vision 模型的 API
* **llama-ocr** ：用于图像中的文本识别功能


##### 如何开始使用

1. 安装 Node.js 和 npm/yarn。
2. 设置环境变量，获取 Together AI API 密钥。
3. 克隆项目并安装依赖项。
4. 启动开发服务器，浏览器访问应用程序。


通过这些步骤，即可轻松使用该工具进行图像到 Markdown 的转换，提升工作效率。
