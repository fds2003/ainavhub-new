## Fireplexity

Fireplexity 是一个由 Firecrawl 的网络抓取 API 驱动的极速 AI 搜索引擎，旨在提供包含实时引用和最新数据的智能答案。其核心功能包括基于 Firecrawl 的实时网页搜索、通过 GPT-4o-mini 提供流式 AI 回复，并为所有 AI 生成的答案提供来源引用以确保信息可靠性。此外，它还能自动展示实时股票数据（通过 TradingView 图表），并通过 AI 生成的智能追问来增强用户交互体验。
