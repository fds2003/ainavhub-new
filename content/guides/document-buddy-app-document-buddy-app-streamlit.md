## Document-Buddy-App

**Document Buddy App** 是一款基于 Streamlit 的强大应用程序，旨在简化文档管理。用户可以上传 PDF 文档，生成高效检索的嵌入，借助智能聊天机器人界面与文档进行交互。

##### 主要功能

* **上传文档** ：用户可以轻松上传并预览 PDF 文档。
* **创建嵌入** ：为文档生成嵌入，以便实现快速搜索和检索。
* **聊天机器人界面** ：用户可以通过智能聊天机器人与文档进行交互，利用所创建的嵌入获取相关信息。
* **联系开发者** ：用户可以通过 GitHub 联系开发者或为项目做出贡献。
* **用户友好的界面** ：应用提供流畅直观的用户界面，结合表情符号和响应式设计，提升用户体验。


##### 使用场景

1. **文档查询** ：用户可以上传专业文档或研究论文，通过聊天机器人快速获取所需信息，提高工作效率。
2. **学习辅助** ：学生可利用此应用提升对学习材料的理解，通过提问获得即时反馈。
3. **文件管理** ：企业和个人用户可以整合所有相关文件，使用智能检索功能快速找到所需资料。
4. **项目协作** ：团队成员可以分享和共同讨论文档内容，利用聊天机器人获得信息支持。


**Document Buddy App** 的设计使其在文档处理和信息检索方面都变得更加高效，适用于各类用户，包括学术研究人员、学生和企业团队。
