## saiki

Saiki 是一个 AI 代理，它允许你使用自然语言来控制你的工具、应用和服务。你可以用自然语言描述你想要做什么，Saiki 会自动判断应该使用哪些工具以及如何使用它们来完成任务。

**主要特点：**

* **开放、可定制、灵活：** 可以通过 MCP 连接各种工具和服务，甚至可以使用自己的 LLM。
* **内置编排：** 自动判断使用哪些工具、以什么顺序调用它们。
* **可定制的界面：** 可以用于 CLI、Web 应用或嵌入到自己的堆栈中。


**使用场景：**

* **代码操作：** 查找代码中的 TODO 注释、创建 React 组件、查看最近提交的文件。
* **开发工作流程：** 启动开发服务器、运行测试、查看 npm 脚本。
* **文件管理：** 查找最近修改的文件、创建新目录、压缩日志文件。
* **GitHub 集成：** 显示未完成的 Pull Request、创建 Issue、检查 CI 状态。
* **自动化工作流：** 例如，根据 README.md 设计着陆页，或总结邮件并将摘要发送到 Slack。
* **构建代理和原型设计：** 自动化各种任务，例如发送邮件，自动发布推文等。


总而言之，Saiki 旨在成为各种工具和服务之间的自然语言接口，方便开发者使用 AI 来自动化任务和构建各种应用。
