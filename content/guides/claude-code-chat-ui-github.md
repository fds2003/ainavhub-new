## Claude Code Chat UI

“Claude Code Chat UI” GitHub项目是一款专为Windows用户设计的、无需WSL依赖的Claude Code原生UI扩展，旨在提供一个更流畅、高效的代码交互环境。其核心功能包括：支持Git Bash和Node.js即可直接运行，提供实时Token用量和费用跟踪，能够自适应多种模块化命令协议(MCP)并支持动态工具查询，兼容包括anyrouter/tuziapi和Kimi K2在内的各类第三方API服务，支持图像粘贴和文件检索以充分利用多模态模型，具备多语言交流及代码注释能力，提供流畅的用户界面以优化“Vibe Coding”体验，并拥有实时的文件操作历史记录及一键撤销/重做功能。该项目通过提供全功能的图形用户界面，解决了Windows环境下Claude Code的路径兼容性问题和终端操作的繁琐性，显著提升了用户体验，并支持通过VS Code Marketplace便捷安装及灵活配置第三方API，尤其强调了首次使用自定义API时需要进行命令行初始化。
