## Gemini CLI UI

“Gemini CLI UI”项目提供了一个跨平台桌面与移动端用户界面，旨在为Google官方的AI辅助编程工具Gemini CLI提供更便捷的图形化操作体验。该UI核心功能包括响应式聊天界面与Gemini CLI无缝通信、集成式Shell终端直接访问、交互式文件与Git浏览器支持实时编辑和版本控制、全面的会话管理及历史追踪、多种Gemini模型选择，以及默认禁用工具以确保安全的“YOLO模式”等特性。它通过可视化项目管理、多格式聊天支持和文件操作，显著提升了用户对Gemini CLI项目的管理与交互效率，所有会话和项目均可持久化并跨设备同步访问，且具备基于SQLite的安全认证系统。
