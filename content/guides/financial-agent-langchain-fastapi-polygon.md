## Financial Agent

Financial Agent是一个基于Langchain和FastAPI构建的金融助手，它通过Polygon API获取股票的当前价格、历史价格、最新消息以及其他金融数据。这个助手的目的是为了帮助用户解决他们的金融相关问题，包括但不限于股票市场的分析。要使用这个助手，用户需要拥有OpenAI和Polygon的API密钥。

##### Financial Agent的功能

Financial Agent可以做以下几件事情：

1. **获取当前价格** ：可以查询某个股票符号的当前价格。
2. **查看历史价格** ：可以查看某个股票符号的历史价格数据。
3. **获取最新新闻** ：可以获取与某个股票符号相关的最新新闻。
4. **获取金融数据** ：可以获取与某个股票符号相关的其他金融数据。


##### 在什么情况下会使用Financial Agent

1. **投资决策** ：在做出买入或卖出股票的决策之前，可以使用Financial Agent来获取最新的市场数据和新闻，帮助做出更为明智的决策。
2. **市场研究** ：在进行市场研究时，可以利用Financial Agent来获取历史价格数据，分析特定股票或市场的趋势。
3. **金融学习** ：对于金融学习者，可以使用Financial Agent来获取实时数据，帮助理解市场动态。
4. **资讯更新** ：对于希望持续跟踪某些股票或市场的最新新闻的用户，Financial Agent可以提供最新的新闻资讯。


##### 使用Financial Agent的两种方式

1. **使用Docker容器部署** ：这种方式更加适合希望在本地安全快速部署的用户。通过Docker，用户只需要一个命令就能构建并运行Financial Agent，便捷且安全。
2. **本地部署（不使用Docker）** ：这种方式适合熟悉Python环境和想要更多自定义设置的用户。通过Python的虚拟环境和Poetry包管理器，用户可以在本地机器上直接运行Financial Agent。


无论是投资者、市场分析师，还是金融学习者，当需要获取和分析股票市场的相关信息时，都可以利用Financial Agent来帮助获取数据和信息，从而做出更为明智的决策或深化对市场的理解。
