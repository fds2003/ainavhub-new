## Enchanted

Enchanted 是一个开源的、兼容 [Ollama](<https://github.com/jmorganca/ollama>) 的优雅应用，适用于 macOS、iOS 和 visionOS，用于与私有托管模型（如 Llama 2、Mistral、Vicuna、Starling 等）进行互动。这款应用的功能类似于 ChatGPT，允许用户连接到自己的私有模型，目的是在 iOS 生态系统（包括 macOS、iOS、Watch 和 Vision Pro）中提供无滤镜、安全、私密和多模态的体验。如果你喜爱这个项目，可以考虑给项目点赞并在 [𝕏](<https://twitter.com/amgauge>) 上关注。

##### 使用场景：

1. **文本对话** ：用于与私有托管的语言模型进行自然语言对话。
2. **多模态交互** ：支持多种输入形式，包括语音和文本，并可附加图像。
3. **隐私保护** ：所有对话历史记录存储在用户设备上，确保数据安全。
4. **设备同步** ：跨设备使用，包括 macOS、iOS、Watch 和 Vision Pro。
5. **离线功能** ：所有功能均可在离线状态下使用。


##### 功能特点：

* 文本到语音（朗读）功能
* 对话历史记录包含在 API 调用中
* 支持深色/浅色模式
* Markdown 支持（美化显示表格、列表、代码块）
* 语音提示
* 图片附件
* 可以指定每个对话的系统提示
* 编辑消息内容或用不同模型提交消息
* 删除单个对话或所有对话
* macOS Spotlight 面板快捷键 `Ctrl`+`⌘`+`K`


##### 使用说明：

1. 下载 Enchanted 应用。
2. 运行 Ollama 服务器并下载模型。
3. 使用 ngrok 转发 Ollama 服务器使其可公开访问。
4. 在 App 设置中指定服务器端点。
