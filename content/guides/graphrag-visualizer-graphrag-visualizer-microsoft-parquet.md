## graphrag-visualizer

GraphRAG Visualizer 是一款用于可视化 Microsoft GraphRAG 产物的应用程序。用户可以上传由 GraphRAG 索引管道生成的 parquet 文件，通过该应用程序轻松查看和分析数据，无需额外的软件或脚本。

##### 主要特点

* **图形可视化** ：支持在“图形可视化”标签中以 2D 或 3D 形式查看图形。
* **数据表展示** ：在“数据表”标签中显示 parquet 文件中的数据。
* **搜索功能** ：支持完整的搜索功能，可以让用户专注于特定的节点或关系。
* **本地处理** ：所有数据处理在用户本地机器上进行，确保数据的安全性和隐私。


##### 使用场景

GraphRAG Visualizer 适合以下场景：

* **数据分析** ：研究人员和数据分析师可以利用该工具快速分析生成的 GraphRAG 数据，洞察数据之间的关系。
* **可视化展示** ：用于向团队或利益相关者展示可视化数据，帮助理解复杂的数据关系。
* **本地开发和测试** ：开发者可以在本地服务器上使用 GraphRAG API 的同时，通过该可视化工具进行数据探索和调试。


通过 GraphRAG Visualizer，用户能够便捷地操作和利用 GraphRAG 生成的数据，提高数据分析和决策的效率。
