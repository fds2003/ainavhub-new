## Coding Agent

Coding Agent Template是一个AI驱动的编码代理项目，旨在自动化代码仓库中的编程任务。它支持Claude Code、OpenAI Codex CLI、Cursor CLI和opencode等多种AI代理进行任务执行，并利用Vercel Sandbox提供安全隔离的代码运行环境。该项目集成了Vercel AI Gateway以实现模型路由和可观察性，并能自动生成描述性的Git分支名称，同时具备任务管理、实时进度更新和基于PostgreSQL的持久化存储功能，所有这些都通过Next.js和Tailwind CSS构建的现代化用户界面呈现，显著简化了AI驱动的开发流程。
