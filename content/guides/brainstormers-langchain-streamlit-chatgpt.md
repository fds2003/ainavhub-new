## Brainstormers

**Brainstormers简介**

Brainstormers是一个基于LangChain和Streamlit构建的工具，旨在提升用户的头脑风暴体验。与直接使用ChatGPT的方式不同，这款应用提供了一系列经过优化的、灵感来源于真实世界的有效头脑风暴技术，使用户能够以结构化的方式探索想法，从而最大化使用大型语言模型（LLM）进行头脑风暴的效益。

该应用支持多种头脑风暴技术，包括：

1. **大思维导图（Big Mind Mapping）** ：适合收集大量想法，扩展思维的范围。
2. **反向头脑风暴（Reverse Brainstorming）** ：识别问题产生的方式，揭示潜在陷阱，促进创新解决方案的生成。
3. **角色风暴（Role Storming）** ：通过不同的视角进行思考，获取多样化的见解。
4. **SCAMPER技术** ：应用替代、组合、调整、修改、作他用、消除、反转等方法来激发独特的想法。
5. **六顶思考帽（Six Thinking Hats）** ：通过六个不同的角度（数据、情感、风险、利益、创造力和过程管理）分析想法。
6. **星爆法（Starbursting）** ：利用“五个W一个H”（谁、什么、在哪里、何时、为什么、如何）生成问题，以深入探索主题。


用户可以通过以下链接直接访问并测试该应用：[Brainstormers Web App](<https://brainstormers-2x7cxamrj2ngjbadgzthi4.streamlit.app/>)。

**使用指南** ：

1. 克隆代码库：`git clone https://github.com/Azzedde/brainstormers`
2. 进入项目目录：`cd brainstormers`
3. 安装依赖：`pip install -r requirements.txt`
4. 运行应用：`streamlit run ./app.py`


**未来展望** ： 该项目的未来改进包括实时互动输出、在HuggingFace上的部署、本地LLM的集成和针对大思维导图的优化运行时。
