## Playwrightess MCP

`Playwrightess MCP`是一个模型上下文协议（MCP）服务器，旨在提供一个持久化的Playwright评估环境，其独特之处在于它暴露了一个具有调用间持久性的JavaScript编程接口，这使得代理能够通过一个统一的`playwright_eval`工具来与Playwright API进行高效交互。该项目目前作为一个实验性工具，并未公开发布。
