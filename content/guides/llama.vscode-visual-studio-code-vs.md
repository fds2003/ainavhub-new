## llama.vscode

##### llama.vscode简介

`llama.vscode` 是一个为 Visual Studio Code (VS Code) 提供本地大型语言模型（LLM）辅助文本补全的扩展。该扩展能够根据用户的输入上下文实时提供智能建议，提升编码和文本编辑的效率。

###### 主要功能

* **实时自动建议** ：在光标移动时自动提供建议。
* **便捷的建议采纳** ：通过 `Tab` 接受建议，`Shift + Tab` 接受建议的第一行，`Ctrl/Cmd + Right` 接受下一个单词。
* **手动切换建议** ：按 `Ctrl + L` 可以手动启用或禁用建议。
* **可控的文本生成时间** ：用户可调整最大文本生成时间。
* **灵活的上下文配置** ：可以根据光标周围的上下文来配置建议范围。
* **支持大范围上下文** ：在低配置硬件上通过智能上下文重用，支持非常大的上下文。
* **性能统计** ：展示系统性能数据，帮助用户了解扩展的运行状态。


##### 使用场景

`llama.vscode` 适用于多种场景，尤其适合以下类型的用户和任务：

1. **程序员** ：在编写代码时，使用扩展提供的智能代码补全，可以显著提高编码速度和准确性。
2. **文本编辑者** ：对于需要快速生成文本或编辑内容的任务，此扩展可以为用户提供实时的文本建议，帮助快速完成文档编写。
3. **学习和研究** ：学生和研究人员在编写报告或论文时，可以利用该工具生成相关内容或补全思路。
4. **低配置环境** ：支持在资源有限的设备上使用，适合普通用户或使用较旧硬件的开发者。


通过该扩展，用户可以在编写代码或文本的同时获得丰富的上下文建议，提升工作效率。
