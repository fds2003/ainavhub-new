## MiGPT

# MiGPT：智能家居，从未如此贴心 ❤️

MiGPT 通过将小爱音箱、米家智能设备与 ChatGPT 的智能理解能力结合，让你的智能家居不仅能够自动化运作，更能懂你、陪伴你。它不仅仅是一个设备控制平台，而是一个能够与你共同成长的智能家居助手。

## 主要功能

1. **LLM 回答** ：小爱音箱可以基于大型语言模型（如 ChatGPT）回答你的各种问题。
2. **角色扮演** ：小爱音箱可根据设定扮演不同角色，如朋友、伴侣等。
3. **流式响应** ：快速响应你的指令，提供即时服务。
4. **长短期记忆** ：记住与用户的对话，提高互动的自然性和个性化。
5. **自定义 TTS** ：支持自定义语音，提升互动体验。
6. **智能家居 Agent** ：通过情感和情境感知，自动调节设备操作。


## 使用场景

1. **设备控制** ：通过语音指令便捷控制灯光、插座、扫地机器人等智能设备。
2. **个性化陪伴** ：通过角色扮演和长短期记忆，与用户建立感情连接，提供贴心陪伴。
3. **智能问答** ：利用 LLM 回答生活小问题，如天气查询、新闻播报等。
4. **情境感知** ：根据用户的情绪和命令，自动播放音乐、调整灯光等。
5. **快速响应** ：使用流式响应，确保命令和问题得到快速处理。


启动和配置 MiGPT 可以通过 Docker 和 Node.js 两种方式，实现便捷的部署。详细的配置参数和环境变量如小米 ID、OpenAI API 密钥等需要提前设置好。

## 问题解决

用户遇到登录、设备不响应等问题时，可以参考文档中的常见问题部分进行排查和解决。在必要时，可以通过提交 issue 反馈问题。
