## secure-rules-files

这个 GitHub 仓库 `secure-rules-files` 提供了一套 AI 原生的基线规则文件，旨在提高生成代码的安全性。这些规则文件通过编码常见编程语言和框架的安全最佳实践，为代码生成过程提供安全指导。这些规则本身也是通过 AI 生成的，并针对多个流行的 AI 编码助手和工具（如 Cursor, Cline, Claude, Windsurf, Codex, Aider, GitHub Copilot）进行了适配。该项目允许用户贡献新的语言和框架，或者改进用于生成规则文件的 prompt。
