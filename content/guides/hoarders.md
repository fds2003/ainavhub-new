## hoarders

##### Hoarders简介

Hoarders是一款自托管的书签管理应用，以满足数据收集爱好者的需求，并添加了AI元素。它允许用户书签链接、记录简单笔记和存储图片，并具有自动抓取链接标题、描述和图片的功能。同时，它支持将书签分类到不同的列表中，提供全文本搜索，并基于AI（如ChatGPT）自动进行标签分类。此外，Hoarders还支持Chrome插件和Firefox插件，以便于快速添加书签，并有iOS和Android应用程序。该应用优先考虑自托管，并计划支持离线阅读。

##### 使用场景

1. **书签管理** : 用户可以将遇到的有趣链接、文章或工具进行书签保存，方便日后查看。
2. **读后续管理** : 类似于Pocket或其他读后续应用，用户可以保存稍后阅读的内容。
3. **笔记记录** : 用户可以记录简单笔记，方便快速整理和查找信息。
4. **图片存储** : 在线保存图片，便于集中管理和组织。
5. **自动分类与标记** : 使用AI技术自动为内容添加标签，方便搜索和管理。
6. **跨平台支持** : 用户可以通过网页、移动应用及浏览器插件随时随地访问和管理自己的书签和笔记。


##### 核心特性

* **自动抓取** : 自动获取和显示链接的标题、描述和图片。
* **全文本搜索** : 对所有存储内容进行全文搜索，提升查找效率。
* **AI自动标记** : 基于AI的自动标签功能，使得分类和查找更智能。
* **多平台支持** : 提供Chrome插件、Firefox插件和移动应用（iOS和Android）以增强便捷性。
* **自托管** : 用户可以自行托管，增强数据的隐私和安全性。


##### 使用技术栈

* **NextJS** : 用于开发Web应用。
* **Drizzle ORM** : 数据库管理和迁移。
* **NextAuth** : 用户认证。
* **tRPC** : 客户端和服务器之间的通信。
* **Puppeteer** : 用于书签抓取。
* **OpenAI** : 为AI功能提供支持。
* **BullMQ** : 后台任务调度。
* **Meilisearch** : 全文内容搜索。


Hoarders专为那些希望在本地服务器上管理并保存各种在线内容的用户设计，是远离主流托管服务的一个极佳替代选择。
