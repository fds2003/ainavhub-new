#### Microsoft Copilot for Security

微软安全副总裁Vasu Jakkal宣布，Microsoft Copilot for Security（国际版）是全球信息安全领域首个独立的生成式AI解决方案。这个AI解决方案每天处理超过78万亿的安全信号，为企业提供深度见解并指导用户的后续行动。Copilot可以帮助安全和IT专业人员全面了解安全态势，更快地采取行动，并增强团队专业技能。此外，Microsoft Copilot for Security通过两次经济研究显示，能够提高资深安全专业人员的工作效率和准确率。这项AI解决方案支持即用即付模式，推出全球销售，可在全球所有国家/地区使用。它还提供了多种新功能，如自定义指令手册、知识库集成等，旨在帮助用户更有效地协作、了解安全态势并做出响应。Copilot还扩展了端到端保护，与微软多个安全产品集成，提供全面的安全解决方案。
