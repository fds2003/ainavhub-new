## Agent Group Chat

Agent Group Chat是一个模拟多个代理人（agents）之间语言互动的仿真系统，旨在研究语言在人类集体行为中的作用。通过设置不同角色和故事情节，例如遗产争执、法庭辩论、哲学论述和电影角色选角等，Agent Group Chat模拟代理人基于各自角色设定自由交谈的场景，观察代理人展现出预料之外且重要的集体突现行为（emergent behaviors）。该系统可以通过特定的环境设置评估代理人表现行为是否与人类期望相符合，并利用计算对话内容的n-gram Shannon熵来评估环境内的混乱程度。研究发现，在代理人与人类期望高度一致的前提下，促进更广泛的信息交换可确保仿真中多样性中的更大秩序性，从而促进更多意料之外且有意义的突现行为的出现。该项目的代码已开源，线上平台即将开放。

##### 使用场景和目的

Agent Group Chat在以下情况下使用:

1. **研究和教育** : 对于进行人工智能、社会学、语言学等领域研究的学者和学生，Agent Group Chat提供了一个实验平台，用于研究和理解语言如何在群体行为和决策中发挥作用。

2. **创意写作与剧情开发** : 编剧和小说家可以使用Agent Group Chat来模拟不同角色之间的对话和交流，以产生创意灵感或加深对角色之间互动的理解。

3. **娱乐与游戏** : 通过设定具体的角色和背景故事，Agent Group Chat可以作为互动式故事叙述和角色扮演游戏的基础，增加用户参与度和沉浸感。

4. **人机交互研究** : 用于探究和改进人工智能在真实场景中的交互能力和自然语言处理技术，特别是在模拟复杂社会交互和群体决策的环境中。

5. **社会和心理学实验** : 社会学家和心理学家可以利用Agent Group Chat来模拟特定社会场景或团体互动，研究个体和群体行为、社会规范、影响力分布等现象。


##### 功能和特点

* **角色设定** : 用户可以自定义角色的名称、目标、背景故事、支持角色、信念等多个角色特性，以及代理人使用的AI模型。

* **资源设定** : 系统允许用户定义故事中的关键资源，包括资源的名称、描述、影响力值和拥有者。

* **情节发展** : Agent Group Chat通过模拟角色之间的对话和行为互动，推动故事情节的发展，产生突现行为和结果。

* **人机交互** : 系统支持将代理人的角色替换为真人，允许用户直接参与模拟对话和决策过程。

* **开源和定制** : 项目代码已开源，支持用户根据需要自行定制故事背景、角色设定和资源等。


通过Agent Group Chat, 用户能够深入探索和理解群体语言互动中的复杂性和动态性，为研究人类行为、设计互动故事和提升AI人机交互提供了有价值的工具和视角。
