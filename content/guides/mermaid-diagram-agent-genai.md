## Mermaid Diagram AI Agent

##### Mermaid Diagram AI Agent 简介

**Mermaid Diagram AI Agent** 是一款结合了生成式人工智能（GenAI）和 Mermaid 图表工具的应用程序，旨在帮助用户快速创建和沟通工作内容的图表。该工具通过简单的文本输入，能够在几秒钟内生成各种类型的图表，如流程图、状态图和饼图，从而提升工作效率。

###### 使用步骤

1. **环境搭建** ：

* 安装 `uv`
* 安装依赖项 `uv sync`
* 设置 OpenAI API 密钥为环境变量
* 可选设置其他 API 密钥，并更新 LLM 模型
2. **生成图表** ：

* 支持单次生成、交互式生成及批量生成，通过命令行输入不同的指令来创建所需图表，例如：
* 流程图生成
* 状态图生成
* 饼图生成


##### 使用场景

Mermaid Diagram AI Agent 的使用场景广泛，主要包括：

* **项目管理** ：快速创建项目进度图或任务分配图，帮助团队成员更好地理解工作流程。
* **教学与培训** ：用于生成教学材料中的图表，帮助学生和学员更容易理解复杂概念。
* **文档编写** ：在技术文档或报告中插入视觉图表，以增强信息的传达效果。
* **产品规划** ：为产品设计和开发流程创建可视化图表，帮助团队在规划阶段明确各个步骤。
* **数据可视化** ：将数据以图表的形式呈现，方便进行数据分析和报告展示。


此工具不仅提高了工作和沟通的效率，还使复杂信息的传达变得更加直观和易懂。通过简单的命令行操作，用户能够快速生成所需的各种图表，极大地方便了日常工作中的视觉表达需求。
