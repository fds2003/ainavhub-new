## Async

Async项目是一个开源的开发者工具，旨在为经验丰富的开发者提供一个结合AI编码、任务管理和代码审查的统一工作流。它能够利用AI（如Claude Code等模型）自动分析代码库、研究编码任务并提出澄清问题，确保在执行前进行充分规划。项目在隔离的云环境（Google Cloud Run）中异步执行代码更改，有效避免了本地环境的干扰和上下文切换。其核心功能包括将复杂的任务拆分为可审查的堆栈差异（stack diffs），实现高效的代码审查和迭代，并无缝管理从GitHub Issue创建到合并拉取请求的整个开发生命周期。通过自动导入GitHub Issue，Async简化了任务跟踪，并提供了一个可视化的Web界面来管理任务和筛选结果，同时支持多任务并发监控和即时通知，高度可定制化的筛选条件进一步提升了其实用性。
