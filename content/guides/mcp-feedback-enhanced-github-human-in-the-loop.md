## MCP Feedback Enhanced

作为一名专业的GitHub仓库分析师，我对`MCP Feedback Enhanced`项目进行了分析，总结如下：

该项目是一个实现了AI辅助开发工具中**人机循环（human-in-the-loop）**工作流程的MCP (Model Context Protocol) 服务器。通过引导AI与用户确认，而非进行推测性操作，它能够将多达25个工具调用整合为一个面向反馈的请求，从而大幅降低平台成本。该项目支持Cursor, Cline 和 Windsurf 等平台，并提供双界面系统 (Qt GUI 和 Web UI)，自动检测环境并选择最佳界面。该项目支持多种图片格式上传，并进行智能压缩，以及支持包括英文、简体中文和繁体中文在内的多语言。

以下是该项目的主要功能、核心要点和关键特性：

1. **人机循环工作流** : 引导AI与用户确认，而非进行推测性操作。
2. **平台支持** : 支持Cursor、Cline和Windsurf等平台。
3. **双界面系统** : 提供Qt GUI和Web UI，分别适用于本地和远程SSH环境。
4. **智能切换** : 自动检测环境并选择最佳界面。
5. **图片支持** : 支持多种图片格式上传，并进行智能压缩。
6. **多语言支持** : 支持英文、简体中文和繁体中文，并可以自动检测系统语言并切换。
7. **模块化架构** : GUI和Web UI都采用模块化设计，易于维护。
8. **集中管理** : 重新组织文件夹结构，方便维护。
9. **现代主题** : 改善视觉设计和用户体验。
10. **响应式布局** : 适应不同的屏幕尺寸和窗口尺寸。
11. **版本更新** : 持续更新，修复bug，并优化功能。
