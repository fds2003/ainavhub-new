## ASTRA

ASTRA 是一个高度可定制的平台，简化了 AI 代理的开发。即使没有任何编码知识，也可以轻松创建高效的多模态 AI 代理。通过 ASTRA，你能够创建直观和无缝的对话互动。

##### 使用场景

1. **语音代理展示** : 利用 ASTRA 可以创建语音代理，实现自然语言处理和语音交互。
2. **本地运行语音代理** : 提供了 Docker 镜像，用户可以在本地快速搭建和运行语音代理，支持 macOS 和 Windows。
3. **代理定制化** : 用户可以通过更改扩展的源代码来定制代理，包括设置不同的语言模型、语音合成和语音识别服务。
4. **多功能扩展** : ASTRA 支持多种扩展，如 OpenAI 的 GPT 系列模型用于自然语言处理，Azure 和 ElevenLabs 的文本转语音服务，Agora RTC 用于低延迟传输等。


##### 定制化步骤

1. **准备服务和密钥** : 如 Agora App ID、Azure 的语音服务 API 密钥、OpenAI API 密钥等。
2. **运行Docker容器** : 将当前文件夹挂载为工作目录，构建并运行代理服务。
3. **修改配置** : 配置 manifest 文件，定制各种服务参数和提示词。

通过 ASTRA，你可以创造出个性化的 AI 语音代理，并进行深度定制，以满足各种应用需求。
