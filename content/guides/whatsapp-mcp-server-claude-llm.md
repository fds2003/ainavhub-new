## WhatsApp MCP Server

WhatsApp MCP Server 是一个工具，它允许像 Claude 这样的大型语言模型 (LLM) 直接访问和操作你的个人 WhatsApp 账户。它通过 WhatsApp Web 多设备 API 连接，将你的消息存储在本地 SQLite 数据库中，并在需要时才将其提供给 LLM。

**主要功能:**

* **搜索消息:** 允许 LLM 搜索你的 WhatsApp 消息。
* **搜索联系人:** 允许 LLM 搜索你的 WhatsApp 联系人。
* **发送消息:** 允许 LLM 向个人或群组发送 WhatsApp 消息。


**使用场景:**

* **AI 助手:** 将 Claude 或类似 LLM 集成到 WhatsApp，以自动化任务，提供智能回复，总结对话等等。
* **消息分析:** 使用 LLM 分析你的 WhatsApp 消息，例如识别主题、情感分析等等。
* **自动化工作流程:** 构建基于 WhatsApp 消息的自动化工作流程。
* **知识库:** 利用 WhatsApp 消息作为 LLM 的上下文数据，使其能更好地理解和回答问题。


**总结:**

WhatsApp MCP Server 提供了一个桥梁，允许 LLM 访问和操作你的 WhatsApp 数据，从而开启了利用 AI 增强 WhatsApp 使用的各种可能性。
