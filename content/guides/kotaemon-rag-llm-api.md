## kotaemon

**Kotaemon简介**

Kotaemon是一个开源的文档问答（RAG）用户界面，旨在为终端用户和开发者提供干净且可定制的聊天解决方案。它支持多种大型语言模型（LLM）API，并且也能运行为本地LLM，从而满足不同用户的需求。

**主要功能：**

* 提供简洁的用户界面，支持多用户登录，用户可以组织文件并进行文档QA。
* 支持多种大型语言模型，包括OpenAI、Azure OpenAI等，以及本地部署的LLM。
* 具备混合RAG管道功能，结合全文本与向量检索来提升检索质量。
* 支持多模态文档问答，可以处理包含图表的文档。
* 提供引用功能，确保生成内容的准确性，并在浏览器中直接查看引文。
* 支持复杂推理，通过问题分解来处理多步问题。


**使用场景：**

1. **终端用户：** 需要对文档内容进行快速问答，获取具体信息时，可以使用Kotaemon提供的界面与文档进行互动。
2. **开发者：** 可以利用Kotaemon构建自己的文档问答管道，集成不同的LLM和数据存储解决方案，定制符合自身需求的功能。
3. **研究学者或数据分析师：** 需要从海量文献中提取信息时，可以使用Kotaemon提高查找效率和准确性。
4. **组织内部知识管理：** 企业内部可以搭建该系统以便员工高效查询内部文档资料。


Kotaemon希望借助其灵活性与可定制性，帮助用户轻松获取所需信息，同时为开发人员提供一个强大的构建平台。
