#### Le Chat Mistral

Le Chat Mistral 是一个新的对话助手，是 Mistral 模型的首次演示，展示了您可以基于 Mistral 模型构建什么，以及您可以在业务环境中部署什么。它是一个多语言对话助手，是与 Mistral AI 各种模型进行交互的对话入口。它提供了一种教育性和有趣的方式来探索 Mistral AI 技术。Le Chat 可以在底层使用 Mistral Large 或 Mistral Small，或者一个名为 Mistral Next 的原型模型，旨在简洁明了。我们正在努力使我们的模型尽可能有用，并且尽可能少地表达观点，尽管仍有很多改进的空间！通过可调的系统级调节机制，当您将对话引向可能产生敏感或有争议内容的方向时，Le Chat 会以非侵入的方式提醒您（有关我们的调节方法，请参阅我们的服务条款了解更多）。对于企业，我们推出了 Le Chat Enterprise，旨在通过自部署能力和精细化的调节机制提升团队的生产力。如果您对此感兴趣，请联系我们的团队。成为早期采用者！我们很乐意听取您的反馈！立即注册成为 beta 用户，让我们知道 Le Chat 中出现的问题，以及它何时可以更合理地思考，或者何时它不如它应该的那样有帮助或不偏袒，我们会加以改进。请注意，目前 Le Chat 无法访问互联网进行讨论，因此它的信息可能不准确，或者回答问题时可能使用过时的信息。
