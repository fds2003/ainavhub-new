## LLM Agent Trader

LLM Agent Trader是一个AI驱动的股票交易回测系统，其核心功能在于整合大型语言模型（LLMs）进行智能交易决策分析。该系统采用Next.js作为前端界面，FastAPI作为后端API网关，并配备了LLM流式回测引擎，能从YFinance等服务获取股票数据，并将回测日志存入SQLite数据库。项目亮点是其LLM智能策略模块，该模块集成了Azure OpenAI（如GPT-4）或Google Gemini等AI模型、技术分析引擎以及风险管理功能，能够自动生成交易信号、计算性能并记录结果，同时支持用户灵活选择不同的LLM提供商。
