## Computer Use - OOTB

##### Computer Use - OOTB 简介

Computer Use - OOTB（开箱即用的计算机使用）是Claude的新计算机使用API的解决方案，方便用户在各种平台上使用，当前已在Windows上进行了测试。该项目基于Gradio提供用户友好的界面，无需Docker即可安装和运行。

###### 更新内容

在10月25日的更新中，该项目新增了通过移动设备远程控制计算机的功能，并且不需要安装任何移动应用程序。

##### 使用场景

1. **远程控制** ：用户可以通过手机浏览器访问界面，远程控制已安装此软件的计算机，允许用户随时随地执行计算机任务。

2. **移动和桌面接口** ：用户可以在手机或计算机上通过简单易用的Gradio界面下发命令，控制计算机执行特定操作，适合需要灵活办公的场景。

3. **支持多平台** ：虽然当前主要支持Windows，但理论上可以在任何平台上运行，适合多种操作系统用户。

4. **AI助手** ：结合Anthropic的API，用户可以通过简单的命令来实现AI对计算机的多种操作，提高工作效率。


##### 注意事项

* 用户需注意AI模型生成的命令可能存在意外或潜在危险，建议持续监控AI的操作。
* 每个任务的费用可能会比较高，因此需要对成本进行控制。


##### 总结

Computer Use - OOTB 提供了一个创新的平台，允许用户通过简单的界面实现对计算机的远程控制，适合现代数字化办公环境，且随着功能不断增加，未来有望支持更多设备和操作系统。
