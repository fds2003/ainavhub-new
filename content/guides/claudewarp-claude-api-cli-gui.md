## ClaudeWarp

ClaudeWarp是一款功能强大的Claude API中转站管理工具，提供CLI和GUI双模式操作，使用户能够轻松添加、切换和管理多个Claude代理服务。其核心亮点在于基于TOML的智能配置管理，支持多代理标签化分类，并能便捷地将API密钥和基础URL导出为多种Shell格式的环境变量。项目还具备美观的PySide6图形界面及Pydantic数据验证，确保配置的准确性与用户体验，旨在高效便捷地管理并灵活配置Claude API的访问。
