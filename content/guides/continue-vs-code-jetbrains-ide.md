## Continue

Continue 是一个领先的开源 AI 代码助手，旨在帮助开发者提高编码效率。用户可以将各种模型和上下文连接到 Continue，从而在 VS Code 和 JetBrains 等开发环境中构建自定义的自动完成功能和聊天体验。

##### 使用场景

1. **聊天功能** ：开发者可以在 IDE 内部随时向大型语言模型（LLM）寻求帮助，而无需离开当前工作环境。

2. **自动完成** ：在编码时，Continue 提供即时的代码建议，减少键入工作量，提高编码效率。

3. **代码编辑** ：允许开发者在当前文件内轻松修改代码，提升编辑的便利性。

4. **常用操作** ：提供快捷方式以执行常见的编码任务，进一步优化工作流程。


总之，Continue 旨在提升开发者的工作效率，使编码过程更加顺畅和高效。
