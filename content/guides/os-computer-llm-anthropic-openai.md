## OS AI Computer Use

OS AI Computer Use 项目是一个本地桌面自动化代理，它通过抽象的LLM客户端（目前支持Anthropic Computer Use，并计划扩展到OpenAI）来根据自然语言任务控制操作系统。其设计实现了LLM供应商无关性，并提供了适用于macOS和Windows的操作系统无关执行层。该工具的核心功能包括具有缓动和时序控制的平滑鼠标移动、点击和拖放，以及通过Quartz或PyAutoGUI实现的可靠键盘输入（包括热键）和屏幕截图功能。它还提供详细的日志记录、运行成本估算、可通过命令行界面使用的独立可执行文件打包能力，并支持通过WebSocket和JSON-RPC与Flutter等UI进行集成，以实现命令与截图、日志流的交互。
