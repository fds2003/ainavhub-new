## Groq App Generator

##### Groq App Generator 简介

Groq App Generator 是一个互动式网页应用，利用 Groq 的 LLM API 生成和修改网页应用。该应用基于 Next.js 和 TypeScript 开发，旨在简化网页应用的创建过程，实现用户友好的互动体验。

##### 主要功能

* **实时应用生成** ：用户通过自然语言查询可以即时生成应用程序。
* **内容安全检查** ：集成 LlamaGuard 进行内容安全性检查，确保生成的内容符合规定。
* **互动反馈系统** ：用户可以通过反馈进行迭代改进，优化应用生成效果。
* **版本控制及历史追踪** ：支持版本管理，使用户能够追踪和恢复之前的应用版本。
* **分享与导出功能** ：用户可轻松分享和导出生成的应用。


##### 技术栈

* Next.js 14 (App Router)
* TypeScript
* Groq SDK
* React Syntax Highlighter
* UUID（用于会话管理）


##### 使用场景

Groq App Generator 适用于多种场景，包括但不限于：

1. **快速原型开发** ：开发者和设计师可以快速生成应用原型，从而加快开发进程。
2. **教育和培训** ：适用于教学场景，帮助学生和初学者了解网页应用的创建过程。
3. **小型项目和个人项目** ：个人开发者可利用该工具轻松创建并修改自己的小型应用。
4. **内容创作者** ：为内容创作者提供便利，生成适合自己需求的应用，无需编码背景。


##### 开始使用

1. 克隆代码库
2. 安装依赖：`npm install`
3. 设置环境变量
4. 运行开发服务器：`npm run dev`


应用将在 `http://localhost:3000` 可用。
