## MindGeniusAI

##### MindGeniusAI 简介

MindGeniusAI 是一个依托 ChatGPT 的思维导图自动生成工具。它的功能主要包括自动生成思维导图、手动添加节点、删除多余节点、修改节点、以 Markdown 格式渲染和显示结果、导出为图片或 JSON 文件、支持单个节点的 AI 生成内容、编辑现有节点、应用内聊天支持以及文件导入功能。

##### 特性

* 基于结果生成思维导图
* 手动添加、删除和修改节点
* 以 Markdown 渲染并显示结果
* 支持导出思维导图为图片或 JSON 文件
* 支持为单个节点生成 AI 内容
* 应用内即时聊天功能
* 导入文件创建新思维导图
* 添加笔记并将其转换为思维导图


##### 使用场景

1. **教育和学习** ：学生可以用其整理和组织笔记，快速生成知识框架。
2. **项目管理** ：项目经理能轻松创建项目计划、任务分解以及团队协作图。
3. **会议记录** ：在会议中实时记录和组织讨论内容，提高效率。
4. **头脑风暴** ：帮助团队快速整理和组织想法，增强创意的呈现和讨论。
5. **个人计划** ：用户可用其规划个人目标、行程安排等。


##### 动机和灵感

* 自动生成基于 ChatGPT 的思维导图，受 Flowise 和 Xmind 启发。
