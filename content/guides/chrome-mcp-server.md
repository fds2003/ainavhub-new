## Chrome MCP Server

Chrome MCP Server 是一个基于 Chrome 扩展程序的模型上下文协议 (MCP) 服务器，旨在将 Chrome 浏览器转变为智能助手，使 AI 能够控制浏览器进行自动化操作、内容分析和语义搜索。它利用用户现有的浏览器环境，如配置、登录状态等，使大型模型或聊天机器人能够控制浏览器，成为日常助手。

该项目的核心功能和关键特性包括：

1. **兼容性广** : 支持任何 LLM、聊天机器人客户端或代理来自动化浏览器。
2. **现有浏览器集成** : 无缝集成现有浏览器环境 (配置，登录状态等)。
3. **完全本地化** : 纯本地 MCP 服务器，确保用户隐私。
4. **流式 HTTP** : 使用流式 HTTP 连接方法。
5. **跨标签页** : 支持跨标签页上下文。
6. **语义搜索** : 内置向量数据库，实现智能浏览器标签内容发现。
7. **智能内容分析** : AI 驱动的文本提取和相似度匹配。
8. **工具丰富** : 提供超过 20 种工具，包括屏幕截图、网络监控、交互操作、书签管理、浏览历史等。
9. **SIMD 加速** : 使用 WebAssembly SIMD 优化，向量操作速度提升 4-8 倍。
