## Artifacts for ChatGPT

##### Artifacts for ChatGPT 简介

Artifacts for ChatGPT 是一款能够即时预览和部署由 ChatGPT 生成的“工件”（artifacts）的 Chrome 浏览器扩展插件。

通过该扩展，用户可以：

* 实时监控浏览器中的 DOM，从 ChatGPT 中获取工件。
* 在扩展侧边栏中预览这些工件（目前仅支持 HTML 格式）。
* 将 HTML 文件随机部署到 Zeabur 平台。


##### 使用场景

1. **开发和测试** ：开发者可以快速预览和部署由 ChatGPT 生成的代码片段，进行即时测试。
2. **内容展示** ：内容创作者可以利用 ChatGPT 生成的 HTML 内容，快速上线并分享给受众。
3. **教学和培训** ：教师可以使用该工具展示 ChatGPT 生成的例子，帮助学生理解编程和网页开发。


##### 未来发展路线

* 将支持 React 和其他 Web 框架。
* 允许使用自定义域名进行部署。
* 将扩展发布至 Chrome 网上应用店（已完成）。
* 支持其他浏览器。
