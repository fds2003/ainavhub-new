## Azure DevOps MCP Server

该项目是Azure DevOps MCP Server，旨在通过本地服务器将Azure DevOps的功能集成到代码编辑器（如VS Code和VS Code Insiders）中，从而方便用户直接在编辑器中执行各种Azure DevOps任务。该项目目前处于公开预览阶段。

其主要功能和特点包括：

1. **核心功能** : 为代理提供Azure DevOps上下文，支持通过自然语言提示词执行如列出项目、构建、发布、代码仓库、测试计划、团队和工作项等操作。
2. **支持的工具** : 提供与Azure DevOps核心服务、工作管理、工作项、代码仓库、构建、发布和测试计划的交互功能。
3. **多种安装方式** : 提供简便的一键安装方式和从公共源安装方法，同时也支持从源码安装（适用于开发者）。
4. **集成GitHub Copilot** : 能够与GitHub Copilot集成，通过Agent Mode，允许用户在代码编辑器内通过Copilot Chat与Azure DevOps MCP Server进行交互。
5. **详细文档** : 提供安装、使用、问题排查和贡献指南等详细文档，方便用户上手和参与项目开发。
