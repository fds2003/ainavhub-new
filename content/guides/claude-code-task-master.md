## Claude Code

Claude Code 是一个综合性的AI代理系统，旨在通过专业化的AI代理、高级编排、组织记忆和创新的Task Master集成，彻底改变开发工作流程并提升代码质量。该系统能够根据自然语言描述，智能协调多个涵盖广泛技术栈（包括前端、后端、基础设施）的专业代理，实现代码库感知的自主开发，有效减少30-40%的开发时间。其核心功能包括智能引导（Bootstrap），可对项目进行一次性分析并自动配置最优代理与工作流；内置工程卓越特性，确保生产级代码质量，如故障容忍、结构化日志和自动化质量检查；以及通过MCP集成实现跨项目学习和知识积累，支持并行执行、顺序推理等专业级工作流，并自动进行提交归因和质量门控，最终实现更高效、更一致、更高质量的软件交付。
