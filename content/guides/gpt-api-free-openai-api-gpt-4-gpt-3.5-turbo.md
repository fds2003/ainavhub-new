## GPT-API-free

**GPT-API-free** 是一个支持多种 OpenAI 模型的 API 平台，包括 **GPT-4** 、GPT-3.5-Turbo、GPT-3.5-Turbo-16K、嵌入模型（embeddings）、DALL·E、Whisper 和 text-davinci。它提供国内动态加速服务，用户无需代理即可直连使用。

##### 主要特点

1. **多模型支持** ：包括 GPT-4、GPT-3.5、DALL·E 和 Whisper 等多种模型，免费版支持 GPT-4（每天限三次）。
2. **高兼容性** ：与 OpenAI 官方接口标准一致，兼容多种软件和插件。
3. **流式响应** ：支持流式数据传输。
4. **国内优化** ：国内线路动态加速，无需科学上网。
5. **保护隐私** ：不收集用户输入或输出数据，内部评估测试使用。
6. **免费和付费版本** ：免费版本提供基本服务，付费版本提供更稳定、快速和不限次使用的服务。


##### 使用场景

1. **个人非商业用途** ：适用于个人用户的日常使用，如撰写文本、代码生成等。
2. **教育和科研** ：适用于教育和非营利性科研项目，免费版支持基础模型调用。
3. **插件集成** ：支持与多种第三方工具和插件集成，如 Zotero 插件、浏览器插件 ChatGPT Sidebar，Jetbrains 插件等。
4. **开发者应用** ：开发者可以使用 API 集成到自己的应用中，如 AutoGPT, gpt_academic, langchain 等。


##### 如何使用

1. **申请免费 API Key** ：[申请内测免费 Key](<https://api.chatanywhere.org/v1/oauth/free/github/render>)。
2. **购买付费 API Key** ：[购买付费 Key](<https://buyca.shop/>) 以获取更高调用限额和更稳定的服务。
3. **国内转发 Host** ：使用以下 Host 进行 API 请求：
* `https://api.chatanywhere.tech`（国内中转）
* `https://api.chatanywhere.com.cn`（国内中转）
* `https://api.chatanywhere.cn`（国外使用，国内需代理）


##### 注意事项

* 免费 API Key 限制每天 200 请求/天/IP&Key，禁止用于商业用途或大规模训练商用模型。
* 服务状态及公告可以在 [status.chatanywhere.tech](<https://status.chatanywhere.tech/>) 查询。
* 付费服务提供更高的稳定性和完整功能支持，收费方式灵活。
