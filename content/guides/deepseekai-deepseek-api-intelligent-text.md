## DeepSeekAI

**DeepSeekAI - 智能网页助手**

DeepSeekAI 是一款基于 DeepSeek API 的非官方浏览器扩展，旨在通过智能交互提升网页浏览体验。用户只需简单地选择文本，即可瞬间获得基于 AI 的响应，从而使浏览更加高效和智能。

##### 核心功能

1. **智能互动** ：

* Intelligent Text Analysis：支持用户对网页文本进行即刻分析。
* Multi-turn Dialogue：支持多轮对话交互。
* Quick Access：用户可通过文本选择、右键菜单及快捷键轻松调出聊天窗口。
* Streaming Response：实时流式显示 AI 回复。
* Model Selection：用户可选择 DeepSeek V3 或 DeepSeek R1 模型。
2. **用户体验** ：

* Draggable Interface：聊天窗口可自由拖动和调整大小。
* Window Memory：记住聊天窗口的大小和位置。
* One-click Copy：轻松复制回应内容。
* Regenerate：支持重新生成 AI 回复。
3. **内容展示** ：

* Markdown Rendering：支持丰富的 Markdown 格式，包含代码块、列表和数学公式。
* Code Highlighting：多种编程语言的语法高亮支持。
* Multi-language Support：支持中英文界面及自动语言识别的 AI 回复。
* Dark Mode：依据系统偏好自动支持暗黑模式。


##### 使用场景

* **学习辅助** ：学生和研究人员可以通过选取网页上的段落，快速获得相关的解读和补充信息。
* **编程支持** ：程序员可以选中代码并通过 AI 获取优化建议或调试帮助。
* **文本分析** ：内容创作者能够快速分析材料并获取灵感，从而优化写作内容。
* **实时翻译** ：用户能够即时翻译选中的文本，促进多语言交流。
* **研究支持** ：学者在浏览科研文章时，可快速提取信息和进行深入探讨。


DeepSeekAI 通过简单、直观的方式，结合强大的 AI 功能，大幅提升了在线工作的效率和体验。
