## 语析：基于大模型的智能知识库问答系统

**语析是一个集成了大语言模型（LLM）、知识库和知识图谱的问答系统，旨在提供强大且灵活的智能问答能力。** 它利用了检索增强生成（RAG）技术，将大模型的能力与特定领域的知识相结合，以提供更准确、更全面的答案。

**核心功能和特点：**

* **多模型支持：** 可以对接多种大模型平台，包括 OpenAI，国内主流大模型，以及本地部署的 vllm 和 ollama。
* **灵活的知识库：** 支持多种文档格式（PDF、TXT、MD等），方便用户导入和管理自己的知识。
* **知识图谱集成：** 利用 Neo4j 知识图谱，可以回答基于知识关联的问题。
* **易于配置：** 通过配置 API 密钥即可快速上手。
* **智能体拓展：** 支持自定义智能体，扩展系统功能（开发中）。
* **支持网页检索:** 集成网页检索功能，能够从互联网上获取最新信息。
* **支持多种向量模型与重排序模型:** 允许用户选择合适的向量模型和重排序模型以优化检索效果。


**使用场景：**

语析适用于需要基于特定知识库或知识图谱进行问答的各种场景，例如：

* **企业内部知识管理：** 将企业文档、手册等导入语析，员工可以通过自然语言提问快速获取所需信息。
* **学术研究：** 将论文、报告等导入语析，研究人员可以更高效地进行文献检索和知识发现。
* **教育培训：** 创建特定领域的知识库，学生可以通过问答的方式学习和巩固知识。
* **客户服务：** 构建智能客服系统，自动回答客户常见问题。
* **智能助手：** 开发个性化的智能助手，提供基于知识的智能问答服务。


**简单来说，语析是一个可以让你用自然语言与你的知识库对话的强大工具。** 它将大模型的智能与你的数据相结合，为你提供更便捷、更高效的信息获取和知识管理方式。
