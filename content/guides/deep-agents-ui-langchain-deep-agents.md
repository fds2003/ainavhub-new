## Deep Agents UI

Deep Agents UI是LangChain `deep-agents` 包的配套用户界面，旨在提供一个可视化平台，用于与处理各种复杂任务的通用AI代理进行交互和管理，其核心功能是连接本地或生产环境的LangGraph服务器，并通过配置部署URL和代理ID等关键参数，实现对深度代理的测试与操作。
