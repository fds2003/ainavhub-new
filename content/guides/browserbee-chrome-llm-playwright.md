## BrowserBee

BrowserBee 是一个开源的 Chrome 浏览器扩展，它允许你使用自然语言控制浏览器。它结合了大型语言模型（LLM）解析指令和规划任务的能力，以及 Playwright 强大的浏览器自动化功能来完成各种任务。

**主要特点：**

* **隐私优先：** 除了LLM之外，BrowserBee 完全在你的浏览器内运行，可以直接与已登录的网站（例如社交媒体账号或电子邮件）交互，而不会损害安全或需要后端基础设施。
* **支持多种 LLM 提供商：** 包括 Anthropic、OpenAI、Gemini 和 Ollama。
* **成本追踪：** 跟踪 token 的使用和价格，让你了解每个任务的费用。
* **丰富的浏览器工具：** 提供各种浏览器工具，用于交互和理解浏览器状态（导航，标签，交互，观察，鼠标键盘，内存）。
* **Playwright 支持：** 使用 Playwright 作为底层浏览器自动化工具。
* **记忆功能：** 记录有用的工具使用顺序，并将其存储在本地，以提高未来使用的效率。
* **用户批准：** 在进行购买或发布社交媒体更新等操作时，会提示用户批准。


**BrowserBee 的使用场景：**

* **社交媒体管家：** 检查社交媒体帐户，总结通知和消息，并帮助你回复。
* **新闻策展人：** 从你喜欢的新闻来源和博客中收集和总结最新的头条新闻，为你提供快速、个性化的简报。
* **个人助理：** 帮助处理日常任务，例如阅读和发送电子邮件和消息、预订航班、查找产品等。
* **研究助理：** 通过收集和组织信息，协助深入研究公司、职位列表、市场趋势和学术出版物等主题。
* **知识书签和总结：** 快速总结文章，提取关键信息，并保存有用的见解以供以后参考。
* **与任何网站聊天：** 提问、生成摘要、填写表格等。


**简而言之，BrowserBee 旨在成为你浏览器中的 AI 助手，让你能够用自然语言完成各种 Web 任务，且注重隐私和效率。**
