## ComfyUI-MimicBrush

ComfyUI-MimicBrush 是一个针对在 ComfyUI 中使用 MimicBrush 的自定义节点，可以通过参考图像进行图像修复和补全。这个项目允许用户在 ComfyUI 中利用 MimicBrush 的功能进行图片的修补，使处理效果更加真实和自然。

##### 使用方法

1. 在 ComfyUI 的 `custom_nodes` 目录下运行以下命令安装：

git clone https://github.com/AIFSH/ComfyUI-MimicBrush.git
cd ComfyUI-MimicBrush
pip install -r requirements.txt


2. 权重文件会自动从 modelScope 下载。


##### 使用场景

* 图片修复：利用参考图像对损坏或不完整的图片进行自动修补。
* 创意设计：在图像编辑和设计中，快速实现高质量的图像合成和修饰。
* 数字内容制作：在影视、广告和游戏等数字内容创作中，提升细节处理和素材整合的效率。


通过 ComfyUI-MimicBrush，用户可以更加便捷地在 ComfyUI 中实现复杂的图像修复和编辑工作。
