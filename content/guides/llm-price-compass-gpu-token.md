## LLM price compass

**LLM Price Compass 简介**

LLM Price Compass 是一个开源项目，旨在收集不同云服务提供商以及 GPU 上的推理成本基准数据，并将其与其他提供商的固定每 Token 成本进行比较。该工具可以帮助用户选择适合自己需求的 GPU、云服务和提供商，从而降低使用大语言模型（LLM）所需的成本。

**使用场景**

1. **成本比较** ：用户可以使用 LLM Price Compass 来比较不同云服务提供商和硬件配置下的推理成本，从而找到最具性价比的方案。

2. **选型决策** ：在需要部署 LLM 的过程中，技术团队可以根据收集的数据，选择最佳的 GPU 和云服务提供商，以提高性能并降低费用。

3. **性能基准测试** ：研究人员和开发者可以查看不同模型（如 Llama 3.1）的性能数据，进行合理的调优与优化。

4. **预算规划** ：企业在制定 LLM 部署计划时，可以根据推理成本数据进行有效的预算规划，避免不必要的开支。

5. **技术社区反馈** ：项目接受贡献和反馈，用户可以提出问题或改进建议，促进社区共同进步。


通过 LLM Price Compass，用户可以更明智地选择和管理其大语言模型的推理成本，从而提升业务效率和竞争力。
