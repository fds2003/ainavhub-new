## Claude Code Proxy

Claude Code Proxy 是一个双重用途的解决方案，它既充当 Claude Code 请求至 Anthropic API 的透明代理，实现实时监控与拦截，又能通过一个美观的Web界面可视化并分析用户的Claude API对话。其核心功能包括将所有API交互记录到SQLite数据库，提供可搜索的请求历史、请求/响应体检查和会话串联，同时还提供一个实时仪表盘，用于展示请求与响应，并支持深入的对话分析，包括自动提示词评分、最佳实践评估和响应质量度量。用户只需通过简单的环境变量设置，即可将Claude Code的请求路由至此代理进行全面的监控与分析。
