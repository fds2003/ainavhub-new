## agents-ui-kit

`agents-ui-kit`项目是一个专业的UI组件库，专门用于构建复杂且美观的AI代理界面、自主助手和多代理系统。它在优秀的前端库`prompt-kit`和`shadcn/ui`的基础上，不仅保留了`prompt-kit`原有的消息显示、提示输入、流式响应和聊天容器等核心聊天界面功能，还创新性地引入了一系列针对AI代理的专属组件，例如展示代理信息、能力和状态的`AgentCard`，可视化任务队列的`TaskQueue`，显示工具调用的`ToolCall`，以及扩展的`AgentConversation`、`SystemPrompt`、`AgentStatus`和`MultiAgentView`等，旨在提供一套全面且高度定制化的解决方案，以支持开发者构建功能丰富的AI代理前端体验。
