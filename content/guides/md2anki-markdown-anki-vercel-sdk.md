## md2anki

md2anki 是一款开源的网络应用，利用人工智能从 Markdown 文件生成 Anki 闪卡，使得将笔记转化为学习资料变得更加便捷。它基于 Vercel AI SDK、GPT-4o-mini、Next.js 等技术构建。

##### 使用场景

md2anki 适用于多种情况，包括但不限于：

1. **学术学习** ：学生可以将课堂笔记或阅读材料上传，通过自动化生成闪卡，轻松复习重点内容。
2. **语言学习** ：将外语学习资料转化为闪卡，帮助记忆单词或句型。
3. **知识整理** ：研究人员可以整理知识点，生成可用于回顾的闪卡，提升学习效率。
4. **考试准备** ：考生可以将复习资料转化为闪卡形式，提高记忆的有效性。


##### 使用步骤

1. 上传包含要转化内容的 Markdown 文件。
2. 点击“生成闪卡”按钮，启动 AI 生成过程。
3. 审核并编辑生成的闪卡（如有必要）。
4. 下载生成的与 Anki 兼容的 `.apkg` 文件。
5. 将该文件导入 Anki 开始学习。


md2anki 是一个非常实用的工具，尤其适合需要高效整理和复习学习资料的用户。
