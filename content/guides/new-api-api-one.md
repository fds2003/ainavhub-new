## new-api

New API是在One API基础上经过二次开发的一个开源项目，主要用于提供给开发者们更便捷的接入和使用不同的人工智能服务，例如文本生成、图像处理等。通过结合现有的API服务，它能够帮助开发者快速实现各种基于人工智能技术的功能。以下是关于New API用途和功能的通俗总结：

##### 主要改进和特点

1. **全新的UI界面** ：改版后的界面更加现代化、用户友好，提高了使用体验。
2. **支持Midjourney-Proxy接口** ：带来了包括图像生成、变换、融合等多种功能，扩大了API的应用领域。
3. **在线充值功能** ：方便了用户根据自己的需求，灵活地为账户充值，当前支持易支付接口。
4. **额度查询** ：通过key查询使用额度的功能，方便了开发者和用户对使用情况的监控。
5. **支持更多模型** ：如GPT-4、DALL-E 3、TTS-1等多种前沿的人工智能模型，以及特定第三方模型。
6. **模型按次数收费** ：为商业使用提供了更灵活的计费方式。
7. **加权随机模式** ：提供了一种新的负载均衡策略，增加了系统的灵活性和可用性。
8. **数据看板** ：提供直观的数据监控界面，方便管理和分析。
9. **令牌调用权限** ：可设置特定令牌能调用的模型，提高了安全性。


##### 部署方式

New API支持基于Docker的部署，兼容SQLite和MySQL数据库，简化了部署和配置过程。对于不熟悉命令行操作的用户，也提供了使用宝塔面板Docker功能部署的说明，进一步降低了部署的门槛。

##### 使用场景

* **开发人工智能应用** ：当你需要在你的应用中快速集成如自然语言处理、图像识别等人工智能功能时，New API提供一站式的服务接入。
* **个人学习和实验** ：对于个人开发者或者学生，New API是一个很好的学习和实验平台，通过实际调用不同的API服务，可以加深对人工智能技术的理解。
* **企业内部工具** ：企业可以利用New API搭建内部的人工智能服务平台，用于提高生产力，如自动化报告生成、数据分析等。


##### 注意事项

使用New API时需要遵循OpenAI的使用条款以及相关法律法规，确保不用于任何非法用途。同时，根据《生成式人工智能服务管理暂行办法》，请特别注意不对中国地区公众提供未经备案的生成式人工智能服务。

New API是一个功能丰富、易于部署和使用的开源项目，适用于开发者快速实现基于人工智能的应用功能，但使用时需注意合规合法。
