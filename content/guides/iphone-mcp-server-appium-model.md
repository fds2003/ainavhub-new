## iPhone MCP Server

iPhone MCP Server项目是一个基于Appium的Model Context Protocol (MCP) 服务器，其主要功能是自动化iPhone任务，通过可流式传输的HTTP接口实现应用控制、UI交互和屏幕截图捕获。该项目能够获取设备信息、列出已安装应用、捕获高效的JPEG格式屏幕截图、列出UI元素并执行触摸操作（如点击、滑动、文本输入），以及通过Bundle ID启动或切换应用。其核心亮点在于优化了XML输出以减少token使用，并通过环境变量提供了高度定制化能力，特别强调了设备UDID作为必要配置。
