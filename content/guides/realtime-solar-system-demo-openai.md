## Realtime Solar System Demo

##### Realtime Solar System Demo 简介

Realtime Solar System Demo 是一个基于 OpenAI 实时 API 的演示项目，能够通过语音与一个三维场景进行交互，该场景展示了整个太阳系。该项目使用 Next.js 构建，并通过 WebRTC 集成实现实时功能，同时利用函数调用功能触发应用中的各种动作。

用户可以在一个互动的环境中，通过提问和语音指令获得关于太阳系的各种信息，比如行星、卫星和轨道等，并且可以看到相应的动画效果。

##### 使用场景

1. **教育和学习** ：这个演示项目可以作为教育工具，帮助学生和天文爱好者了解太阳系的构成，并通过与模型的互动加深对宇宙知识的理解。

2. **科研支持** ：研究者可以利用这个演示来展示太阳系中的天文学概念，帮助进行科研交流和讨论。

3. **娱乐和互动体验** ：它也可以作为一种娱乐体验，让用户在玩乐中学习，增强参与感和互动性。

4. **开发者工具** ：对于开发者来说，该项目展示了如何使用 OpenAI 的实时 API 和函数调用功能，可以作为开发自定义应用的基础模板。


##### 适用的特定交互

* 询问特定行星将触发对该行星的视觉聚焦。
* 询问关于卫星的问题，会出现相应的卫星图像。
* 针对图表数据的询问会展示相应的数据图表。
* 询问国际空间站的位置信息会在场景中显示出国际空间站的位置。
* 发出结束对话的信号会重置视角。


这种功能丰富、互动性强的应用场景，使得 Realtime Solar System Demo 成为一个多用途的教育和展示工具。
