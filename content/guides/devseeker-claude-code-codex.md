## Devseeker

DevSeeker 是一个受 Claude Code 和 Codex 启发的代码生成和编辑工具。它的主要功能包括：

1. **代码管理** ：可以管理代码文件和文件夹。
2. **代码创建与编辑** ：能够根据命令创建和编辑代码。
3. **短期记忆** ：可以将代码存储在短期记忆中。
4. **代码审查** ：提供代码变更审查功能。
5. **代码运行** ：可以运行代码文件。
6. **Token 消耗计算** ：能够计算 Token 的使用情况。
7. **多种编码模式** ：提供多种编码模式选择，比如标准模式，优化模式，澄清模式，轻量模式，自修复模式。
8. **广泛的模型支持** ：支持多种 AI 模型，并且可以使用本地模型或者 Azure OpenAI。


此外，DevSeeker 还提供多种命令行选项和环境变量配置，方便用户进行定制和调试。
