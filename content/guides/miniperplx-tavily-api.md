## MiniPerplx

MiniPerplx 是一个简约的 AI 驱动搜索引擎，旨在帮助用户在互联网上快速查找信息。它利用最新的人工智能技术，为用户提供精确的答案和信息检索功能。MiniPerplx 的主要特点包括：

* **AI驱动搜索** ：使用 AI 模型回答用户问题。
* **网页搜索** ：通过 Tavily 的 API 进行网页搜索。
* **网址特定搜索** ：从特定网址获取信息。
* **天气查询** ：使用 OpenWeather 的 API 获取任何地点的实时天气。
* **编程支持** ：使用 E2B 的 API 运行多种编程语言的代码片段。
* **地图功能** ：通过 Google Maps、Mapbox 和 TripAdvisor API 获取任何地点的位置。
* **翻译功能** ：使用 Microsoft 的 Translator API 实现多语言翻译。
* **YouTube 搜索** ：搜索 YouTube 视频并获取时间戳及逐字稿。
* **学术搜索** ：查找学术论文。
* **产品搜索** ：在亚马逊上搜索产品。
* **航班追踪** ：使用 AviationStack 的 API 追踪航班。
* **流行影视信息** ：获取流行电影和电视节目的信息。


##### 使用场景

MiniPerplx 可广泛应用于多个场景，例如：

* 学术研究人员可以使用学术搜索功能查找相关论文。
* 开发者可以在编程过程中使用代码片段运行功能。
* 日常用户可以快速获取天气预报、影视信息和产品推荐。
* 商务旅行者可以查询实时航班信息。


总之，MiniPerplx 是一个功能强大且易于使用的搜索工具，适合各种需求的用户。
