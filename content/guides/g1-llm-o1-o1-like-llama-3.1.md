## g1

g1 是一个实验性原型，旨在利用提示策略提升大型语言模型（LLM）的推理能力，创造类似于 o1 的推理链。与 o1 不同，g1 将所有推理步骤显式展示，并使用开源模型。该项目意在激励开源社区探索新策略来生成 o1-like 的推理能力。

g1 使用 Llama-3.1 70b 模型，通过动态的 Chain of Thought（思维链）使 LLM 能够“思考”并解决一些通常让领先模型感到困扰的逻辑问题。每个推理步骤都有标题且对用户可见，系统提示还包含针对 LLM 的建议，例如“探索备选答案”或“至少使用三种方法得出答案”。这种结构使得 LLM 的推理能力得到了显著提升。

##### 使用场景

g1 适用于解决简单逻辑问题，例如“草莓中有多少个 R？”和“0.9 和 0.11 哪个更大？”通过可视化的推理步骤，用户能更好地理解 LLM 的推理过程，这在教育、研究以及需要逻辑推理的多种应用中具有实际价值。

##### 总结

g1 是一个展示增强 LLM 推理能力潜力的工具，尤其是在处理开放性逻辑问题时，通过动态的推理链结构和提示策略，显著提高了针对特定问题的解决效率，展现出开源模型在推理领域的应用重要性。
