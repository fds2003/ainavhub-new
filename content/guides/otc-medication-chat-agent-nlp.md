## OTC Medication Chat Agent

“OTC Medication Chat Agent”项目是一个智能聊天代理，其核心功能是根据用户输入的症状和常见用例，推荐合适的非处方药物。该代理通过自然语言处理（NLP）技术来解析症状，并利用Streamlit构建了简洁易用的用户界面，方便用户进行交互。该项目的重要特点在于其明确指出，它仅供教育和信息参考，不能替代专业的医疗建议。
