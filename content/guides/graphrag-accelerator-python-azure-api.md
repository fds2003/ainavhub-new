## GraphRAG Accelerator

GraphRAG Accelerator 是一个解决方案加速器，它建立在 [graphrag](<https://github.com/microsoft/graphrag>) Python 包之上，并通过在 Azure 上托管的 API 端点进行扩展。这个加速器可以用来触发索引管道，并启用 graphrag 知识图谱的查询功能。

##### 使用场景

1. **企业知识管理** ：通过构建和查询企业内部的知识图谱，有效管理和利用企业知识资源。
2. **数据分析和智能推荐** ：在数据密集型应用中，如电子商务或内容推荐系统，利用知识图谱进行用户行为分析和个性化推荐。
3. **研究和教育** ：在学术研究和教育领域，通过知识图谱检索和组织复杂的研究信息和文献。
4. **医疗健康** ：在医疗健康行业，应用知识图谱进行疾病研究、药物发现和个性化医疗方案的开发。
