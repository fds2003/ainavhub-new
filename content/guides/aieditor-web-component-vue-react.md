## AiEditor

AiEditor 是一个由 AI 驱动的新一代富文本编辑器，基于 Web Component 技术，支持 Vue、React、Angular 等几乎所有前端框架，同时兼容 PC 和移动端设备，并提供明暗两种主题。其主要特点包括灵活的配置和易于使用，允许开发者轻松构建各种文字编辑应用。

主要特色：

1. **AI驱动** ：完全由 AI 驱动，支持对接任何大型模型，包括私有模型，用户可以使用自己的私有大型模型 API 密钥。
2. **友好易用** ：基于 Web Component 开发，易于集成到任何主流前端框架中，采用 LGPL 开源协议。
3. **多个版本** ：除了开源版本，还提供功能更强大的商业版本，不限制用户数量和应用数量。


功能特性包括基础文字编辑、增强功能（如撤销、重做、格式刷）、附件支持、代码编辑、表格操作、Markdown 支持和多种 AI 功能（如 AI 续写、AI 优化、AI 校对、AI 翻译）等。此外还具有国际化、多主题和移动端适配等特性。

官方主页和更多信息请访问：https://aieditor.dev
