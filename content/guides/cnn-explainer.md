## CNN Explainer

**CNN Explainer** 是一个互动式的可视化系统，旨在帮助非专业人士学习卷积神经网络（CNN）。该工具通过生动的视觉和互动演示，使用户更容易理解和掌握CNN的基本概念和运作机制。

主要特点：

1. **互动学习** ：通过动态的可视化图表，展示CNN的层次结构和数据处理过程。
2. **非专业友好** ：设计简单易懂，适合没有编程或深度学习基础的用户。
3. **开源项目** ：可以下载和本地运行，并支持自定义模型和图像类别。


##### 使用场景：

1. **教育** ：作为学习工具，用于教学和自学，帮助学生理解复杂的深度学习模型。
2. **研究** ：科研人员可以利用该工具进行模型分析和可视化展示，提升研究的直观性和说服力。
3. **普及AI知识** ：帮助大众和技术从业者快速入门和理解深度学习的核心概念和应用。


##### 运行步骤：

1. 克隆或下载仓库：

git clone git@github.com:poloclub/cnn-explainer.git
# or for a shallow copy
degit poloclub/cnn-explainer


2. 安装依赖：

npm install


3. 运行CNN Explainer：

npm run dev


4. 在浏览器中访问 [localhost:3000](<http://localhost:3000>) 查看运行情况。


CNN Explainer 是一项由乔治亚理工学院和俄勒冈州立大学合作的研究项目。如果对项目有任何疑问，可以在 [GitHub](<https://github.com/poloclub/cnn-explainer/issues>) 上提出。
