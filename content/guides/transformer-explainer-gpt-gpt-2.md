## Transformer Explainer

Transformer Explainer 是一个交互式可视化工具，旨在帮助用户学习如何使用基于Transformer的模型（如GPT）。该工具在浏览器中运行一个实时的GPT-2模型，让用户可以输入自己的文本，并实时观察Transformer内部组件和操作如何协同工作来预测下一个词元。

##### 使用场景

1. **学习和教学** ：帮助学生和研究人员理解Transformer模型的内部机制，是教育领域的优秀辅助工具。
2. **模型调试** ：开发者可以用它来实时监控模型预测流程，排查和修正模型问题。
3. **科研和开发** ：研究人员可以通过可视化工具探索模型行为，推动自然语言处理技术的发展。


##### 本地运行步骤

1. 安装前提：Node.js 20或更高版本、NPM。
2. 克隆代码库并启动开发服务器：

git clone https://github.com/poloclub/transformer-explainer.git
cd transformer-explainer
npm install
npm run dev


3. 在浏览器中访问 `http://localhost:5173` 即可使用。
