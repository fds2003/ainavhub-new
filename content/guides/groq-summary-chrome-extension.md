## Groq Summary Chrome Extension

Groq Summary Chrome Extension是一款旨在提升网页浏览体验的强大工具，利用Groq技术为用户提供网站内容的摘要。该扩展的主要功能包括：

* **摘要整个网页** ：快速获取任何网站内容的主要信息。
* **摘要选定文本** ：用户可以高亮网页上的特定文本，该扩展会对所选部分进行总结。
* **与内容互动** ：用户可以提出问题或与网页内容进行对话，以加深对内容的理解。


##### 适用场景

Groq Summary Chrome Extension非常适合需要快速理解或与在线内容进行互动的用户，像研究人员、学生以及专业人士等人群。

##### 使用要求

要使用该扩展，用户需提供Groq API密钥，该密钥将安全存储在扩展的本地存储中，以用于身份验证。

##### 主要特点

* **基于React** ：采用React库构建，提供现代高效的用户界面。
* **ShadCN组件** ：利用ShadCN实现一致且可定制的组件样式。
* **暗黑模式支持** ：在低光环境下提供更好的用户体验。
* **热重载** ：在开发时支持热重载，方便无缝编码。


##### 开发设置

用户在本地开发时，可以通过以下步骤设置和运行扩展：

1. 安装依赖并启动开发服务器：

pnpm dev


2. 打开浏览器并访问：

http://localhost:5173


##### 生产构建

为了构建适用于生产环境的扩展，用户可执行以下命令：

1. 运行构建命令：

pnpm build


2. 扩展的弹出窗口将可在以下位置找到：

dist/index.html


总体而言，Groq Summary Chrome Extension是提升网上信息处理效率的实用工具，适合各类需要快速获取信息的用户。
