## Octo

Octo是一个专注于提供友好的零遥测编码助手，其核心功能在于作为多模态LLM（大型语言模型）助手，支持与任何OpenAI或Anthropic兼容的LLM API进行交互，并允许在对话中灵活切换模型以优化响应。该项目通过整合定制的机器学习模型，能够自动处理编码模型中的工具调用和代码编辑失败，显著提升了其智能纠错能力，尤其擅长管理复杂的多轮对话和“思考令牌”以最大限度发挥LLM效能。Octo设计上高度注重用户隐私，不收集遥测数据，并原生支持本地运行的LLM。它还提供了强大的Docker沙盒功能，允许在隔离容器内执行操作，同时通过分层指令文件（如`OCTO.md`）实现高度可定制的项目和全局规则，并支持与MCP服务器连接以获取丰富数据源。
