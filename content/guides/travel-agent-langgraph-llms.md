# AI旅行助手介绍

AI旅行助手是一个智能旅行助手项目，基于LangGraph技术，利用多种语言模型（LLMs）来处理旅行相关任务，如查找航班、预订酒店和发送个性化电子邮件。此助手能够与用户互动，调用必要的工具，提供无缝的旅行规划体验。

## 主要功能

* **状态记忆交互** ：助手能够记住用户的交互，确保用户体验的连贯性。
* **人机协作** ：用户可以控制关键操作，例如在发送电子邮件之前审核旅行计划。
* **动态使用多种LLM** ：助手根据不同任务智能切换不同的语言模型，如工具调用和电子邮件生成。
* **邮件自动化** ：能够自动生成并通过电子邮件发送详细的旅行计划。


## 使用场景

AI旅行助手的应用场景非常广泛，主要包括：

1. **旅行规划** ：用户可以输入具体的旅行需求（如目的地、出发地、日期等），旅行助手会迅速生成包含航班和酒店选项的旅行计划。

2. **航班与酒店搜索** ：助手利用Google航班和酒店API，快速为用户提供合适的航班和酒店选择，并附上相关链接，方便用户点击。

3. **电子邮件发送** ：通过人机协作，旅行助手能够在用户确认信息后，自动发送格式化的旅行信息至用户邮箱，提高用户体验。

4. **个性化推荐** ：根据用户的偏好（如心仪的酒店星级、航空公司等），助手可以提供个性化的旅行推荐，增加用户的满意度。


总之，AI旅行助手致力于简化旅行规划过程，让用户轻松便捷地获得信息及服务，提升整体旅行体验。可以在用户需要航班信息、酒店选择及相关服务时提供实时帮助，是现代数字旅行时代的一个有力助手。
