## Yi-Coder

Yi-Coder 是一系列开源的代码语言模型，具有少于 100 亿个参数的顶尖编码能力。其主要特点包括：

* **长上下文理解** ：Yi-Coder 支持最长 128K 令牌的上下文理解，特别适合处理复杂的编码任务。
* **多语言支持** ：该模型支持 52 种主要编程语言，包括 Java、Python、JavaScript、C++ 等常用语言。


## 使用场景

1. **代码生成** ：能够根据用户输入的自然语言描述生成代码，帮助开发者快速实现功能。
2. **代码补全和修改** ：在编写代码时，可以实时提供代码补全和修改建议，从而提高编码效率。
3. **教育和学习** ：适合用于编程教育，帮助学生理解编程概念和语法。
4. **自然语言查询转 SQL** ：可以将自然语言的问题转换为 SQL 查询，便于与数据库交互。
5. **网页开发** ：根据提纲或需求，自动生成网页代码，简化网页开发流程。
6. **代码质量检查** ：可用于对现有代码的质量进行审查和优化建议。


Yi-Coder 提供了多种使用方式，包括基于 Ollama 的本地运行和通过 transformers 进行的推理，还可通过 vLLM 进行快速推理与服务。该模型的开源使得开发者可以根据自己的需求进行自定义和优化。
