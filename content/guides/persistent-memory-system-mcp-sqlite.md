## Persistent AI Memory System

Persistent AI Memory System 是一个专为AI助手打造的综合记忆系统，它提供持久化、可搜索的存储能力，能够全面追踪多来源的对话历史、记录模型上下文协议（MCP）下的工具调用，并实现智能调度。其核心功能包括基于SQLite的增强记忆系统，支持通过LM Studio嵌入进行语义搜索；具备独立的解析器，能够导入并去重来自ChatGPT、Claude、Ollama、VS Code/GitHub Copilot等多种平台的聊天记录；通过MCP服务器实现对工具使用的实时监控和日志记录，进而支持AI的自我反思；同时，它还能管理时间事件和追踪VS Code项目中的上下文。该系统设计为跨平台兼容，支持零配置快速部署，并提供高度可定制性，旨在帮助AI助手有效整合并利用其在不同平台和交互过程中积累的知识与上下文。
