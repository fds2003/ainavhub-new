## LLM Platform with Meshtastic

##### LLM Platform with Meshtastic 概述

LLM Platform with Meshtastic 是一个将大型语言模型（LLMs）与 Meshtastic 网状通信网络集成的平台。通过该平台，网络中的用户可以与 LLM 互动，以获得简洁且自动化的响应。该平台不仅能与用户进行交流，还能够执行特定任务，例如：

* 拨打紧急服务电话
* 发送消息
* 从传感器获取信息


目前，该平台仅支持紧急服务的演示功能，未来将逐步增加更多工具，用户也可以根据指南添加自己的工具。

##### 主要特点

* Meshtastic 与 LLM 之间的双向通信
* 支持一般广播或针对性的响应
* 对超过 200 个字符的长响应进行自动分块处理
* 保留消息历史以支持上下文相关的互动
* 可以在响应中包含节点特定信息（如电池电量、位置信息等）
* 可以通过 LLM 基于用户的提示执行任务


##### 使用场景

1. **紧急服务响应** ：用户可以通过 Meshtastic 网络发送请求，快速拨打紧急服务电话，以求助或报告紧急情况。
2. **信息获取** ：例如，通过 LLM 向传感器请求实时数据（如温度、湿度等），用户可以在偏远地区或缺乏互联网连接的环境中获取必要的信息。
3. **消息传递** ：用户可以在网状网络中发送和接收消息，尤其是在无法使用传统通信网络的情况下。
4. **上下文呼应的交互** ：通过保留消息历史，用户可以进行更连贯的对话，使互动体验更加自然和高效。


通过这一平台，用户能够在保证网络连通性的前提下，有效利用 LLM 提高通信效率和便利性。
