## Ollama App

Ollama App 是一个现代且易用的 Ollama 客户端，提供良好的用户体验，同时确保所有数据都在本地网络中私密处理。该应用程序并不在设备上托管 Ollama 服务器，而是连接到一个预设的 Ollama 服务器并使用其 API 端点。

##### 使用场景

1. **聊天互动** ：Ollama App 提供一个简洁的聊天界面，用户可以选择不同的模型来进行聊天。该应用程序支持多模态输入，可以通过图片和文本进行互动。
2. **自定义配置** ：用户可以自定义应用的许多设置，如主机地址、行为模式、界面外观等。
3. **数据备份与导出** ：应用支持将聊天内容导出保存，用户可以备份数据以防丢失或者在多个设备间同步。
4. **多语言界面** ：应用内置多语言支持，如英文和德文，用户可以根据需要选择合适的语言界面。
5. **语音交互** ：Ollama App 还提供实验性的语音模式，支持语音输入进行文本转换。


##### 主要功能

* **安装与初始设置** ：通过下载 APK 文件并进行安装，用户可以快速上手。初次使用时需设置主机地址等基本信息。
* **聊天视图** ：核心的聊天界面，支持消息发送、删除和编辑，兼容 markdown 语法。
* **模型选择器** ：可以从已安装的模型中选择聊天使用的模型，并支持多模态模型。
* **边栏菜单** ：提供快速创建新聊天、进入设置等操作，并可以管理已有的聊天。
* **丰富的设置选项** ：用户可以配置主机、自定义请求头、系统提示、界面行为、语音模式和导出功能等。


Ollama App 适用于需要与 AI 进行多方面互动的用户，尤其是注重隐私和本地网络使用的场景。
