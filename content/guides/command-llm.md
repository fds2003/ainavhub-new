## Command R+

# 通俗解释Command R+：为企业设计的可扩展大型语言模型(LLM)

## 简介

Command R+ 是一个针对企业级工作负载优化的先进大型语言模型（LLM），它是基于R系列LLM的最新成员，专为处理真实世界中的企业用例而设计。与之前的Command R模型一样，Command R+ 提供了128k令牌（token）的上下文窗口，并且在以下几个方面表现优异：

* **先进的检索增强生成（RAG）** ：通过引用减少误导性信息出现
* **多语言支持** ：覆盖10种关键语言，支持全球商业运营
* **工具使用** ：自动化复杂的商业流程


Command R+ 的性能在可扩展市场类别中领先，并且在关键的商业能力上与更昂贵的模型竞争。

## 使用场景

你可能会在以下情况下使用Command R+：

* **企业AI部署加速** ：通过微软Azure进行首发，Command R+ 加速企业AI的采用。
* **针对特定信息的精准检索和生成** ：使用RAG减少误导信息，提供更可靠和可验证的解决方案。
* **跨国公司的多语言交流** ：通过支持10种关键语言，帮助公司跨地区、跨文化无缝工作。
* **复杂商务流程自动化** ：通过API和LangChain访问工具使用能力，自动化复杂商务流程。
* **数据隐私和安全** ：保护客户数据，隐私，并确保安全使用AI。


## 性能和特点

Command R+ 在多个方面进行了优化，包括：

* **行业领先的RAG解决方案** ：提供高度可靠、可验证的企业级RAG应用。
* **多步骤工具使用** ：允许模型结合多个工具来完成复杂任务，甚至在遇到工具失败时尝试纠正。
* **多语言支持** ：优化了10种全球商业关键语言的翻译和文本生成能力。


## 成本

与Command R相比，Command R+ 的定价更高，反映了其高级功能和优化。

## 数据隐私和安全承诺

Command R+ 和所有R系列模型都承诺保护客户数据和隐私，为企业用户提供安心的AI使用体验。

## 合作伙伴与评价

Command R+ 已经获得了多家公司的支持和认可，包括与微软Azure、Oracle Cloud Infrastructure (OCI) 的合作，以及行业领袖如Scale AI和Borderless AI的评价。

Command R+ 是为了满足企业级需求而设计的大型语言模型，它凭借强大的性能、先进的技术特点和多语言支持，在多个方面都显示出了巨大的潜力，特别是在数据隐私和安全方面给予企业足够的信心。无论你是想加速AI的部署，优化复杂的商务流程，还是需要支持全球业务运营，Command R+ 都能提供强有力的支持。
