## Bitwarden Server

Bitwarden Server 是一款开源密码管理工具的服务器端组件，提供了所有 Bitwarden 客户端应用程序所需的 API、数据库和核心基础设施。它是使用 C# 和 .NET Core（ASP.NET Core）构建的，数据库使用 T-SQL/SQL Server。该项目可以在 Windows、macOS 和各种 Linux 发行版上跨平台开发、构建、运行和部署。

##### 使用场景

1. **个人密码管理** ：用户可以通过自托管 Bitwarden Server 安全地存储和管理自己的密码、信用卡信息和其他敏感数据。
2. **企业安全** ：企业可以在内部网络中部署 Bitwarden Server，以保护员工的密码和企业敏感信息，增强安全性。
3. **团队协作** ：支持团队使用 Bitwarden 共享密码和加密信息，确保信息的安全传递。
4. **自定义与控制** ：由于是自托管解决方案，用户可以完全控制数据和服务器设置，根据需求进行个性化配置。


通过 Docker，用户可以轻松地在不同平台上部署 Bitwarden Server，适应不同的使用需求和环境。
