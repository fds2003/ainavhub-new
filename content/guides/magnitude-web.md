## Magnitude

Magnitude 是一个开源的 AI 智能体驱动的端到端 Web 测试框架。它允许你使用自然语言构建测试用例，并利用视觉 AI 智能体来执行和适应 Web 界面的变化。

**主要特点:**

* **自然语言测试:** 使用自然语言编写测试步骤和断言，简化测试用例的创建。
* **AI 驱动:** 通过强大的推理智能体来规划和调整测试，以及快速的视觉智能体来可靠地执行测试。
* **灵活的执行:** 可以在本地或 CI/CD 管道中运行测试。
* **托管和自托管:** 可以使用托管服务，也可以自托管，使用自己的浏览器基础设施和 LLM 提供商。


**使用场景:**

* **Web 应用的自动化测试:** 用于自动化 Web 应用的端到端测试，确保应用的各项功能正常运行。
* **回归测试:** 可以帮助快速检测 Web 应用中的回归错误，确保新代码不会破坏现有功能。
* **持续集成/持续交付 (CI/CD):** 可以集成到 CI/CD 流程中，自动化测试 Web 应用，并提供快速反馈。
* **简化测试开发:** 通过自然语言描述测试意图，降低测试脚本编写的复杂度，让测试编写更接近业务需求，测试用例更易读易维护。
