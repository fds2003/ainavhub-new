## ZipAgent

ZipAgent是一个精简高效的Python AI Agent框架，仅通过约700行核心代码便实现了Agent引擎、强大的工具系统和完整的对话管理功能，旨在帮助用户快速构建个性化AI助手。其核心特点包括简洁的API设计、利用`@function_tool`装饰器轻松扩展AI能力的机制、支持实时交互的流式输出、自动化的上下文管理，以及通过Model Context Protocol实现外部工具的无缝集成。该框架现代化且高质量，基于Python 3.10+并支持异步编程，广泛应用于智能客服、代码辅助、数据分析、内容创作、工作流自动化及知识问答等多元场景。
