## Claudia

Claudia 是一款强大的桌面应用程序，旨在提升用户与 Claude Code 的交互体验。它通过图形用户界面(GUI)简化了 Claude Code 会话的管理、自定义代理的创建、使用情况的跟踪等任务。

主要功能和特点：

* **项目与会话管理：** 提供可视化的项目浏览器，方便用户浏览和管理 Claude Code 项目；支持会话历史记录，方便用户查看和恢复过去的编码会话；内置智能搜索，快速查找项目和会话；提供会话洞察，一览会话的初始信息、时间戳和元数据。

* **CC 代理：** 允许创建具有自定义系统提示和行为的专用代理；建立代理库，用于存储不同任务的专用代理；支持在安全沙箱环境中运行代理，具有细粒度的权限控制；详细记录代理运行历史，包括日志和性能指标。

* **高级沙箱：** 提供操作系统级别的安全沙箱(Linux 上的 seccomp、macOS 上的 Seatbelt)；支持创建具有精细访问控制的可重用安全配置文件；实时监控和记录所有安全违规行为；支持跨团队和系统共享沙箱配置文件。

* **使用分析仪表盘：** 实时监控 Claude API 的使用情况和成本；详细分析模型、项目和时间段的 Token 使用情况；通过可视化图表展示使用趋势和模式；支持导出使用数据，用于统计和分析。

* **MCP 服务器管理：** 从中央用户界面管理模型上下文协议(MCP)服务器；通过用户界面或导入现有配置轻松添加服务器；在使用前验证服务器连接；从 Claude Desktop 导入服务器配置。

* **时间线与检查点：** 在编码会话中的任何时间点创建检查点；使用分支时间线浏览会话历史记录；一键恢复到任何检查点；从现有检查点创建新分支；查看检查点之间的具体更改。

* **CLAUDE.md 管理：** 在应用程序内直接编辑 CLAUDE.md 文件；实时预览 Markdown 渲染效果；扫描项目中的所有 CLAUDE.md 文件；提供完整的 Markdown 语法高亮支持。
