**项目名称：OCode**

OCode是一个基于Ollama模型的终端原生AI编码助手，它提供深度代码库智能和自主任务执行。该项目的核心在于将企业级AI辅助直接引入开发工作流程，无需代理即可与本地或远程Ollama模型无缝协作，并通过可扩展的插件层(MCP)实现第三方集成。OCode包含文件操作、文本处理、数据处理、系统操作和开发工具等19个以上的专业工具，支持代码生成与修改、项目理解、开发自动化、数据处理和系统操作等功能。通过增强的会话解析、智能工具选择和性能优化，OCode可以更准确、更高效地完成开发任务。
