## Tabby VSCode Agent

Tabby VSCode Agent 是一个专为 Tabby Terminal 设计的 VS Code 插件，其核心功能是实现 AI 驱动的终端控制与自动化。该项目通过内建的 Model Context Protocol (MCP) 服务器和捆绑的 stdio 服务器，确保与 VS Code 的深度集成，允许 AI 执行命令、读取终端输出、管理 SSH 会话、访问终端缓冲区以及追踪命令历史和输出。它显著优化了在 VS Code 中的用户体验，提供了从导航栏直接开启 Copilot 聊天、流畅的设置与登录流程以及大幅提升的执行速度和响应性。为增强操作的安全性与用户控制力，项目还特别引入了“结对编程模式”，要求 AI 在执行终端命令前获得用户的明确确认。
