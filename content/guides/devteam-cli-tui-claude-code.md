## DevTeam CLI

DevTeam CLI是一个专为本地编码智能体设计的终端工具，通过统一的终端用户界面（TUI），它使得用户能够并行管理和协调多个Claude Code、Codex或Gemini智能体进行开发。其核心功能包括利用Git worktrees启动多任务并行开发，提供内置的差异查看器以审阅智能体提交的代码更改并直接添加反馈，实时高亮显示需要用户介入的智能体，详细展示每个智能体的工作进度（如代码差异行数、功能推送及GitHub PR状态），并支持在各自工作树中运行程序以便快速测试。用户可以为不同功能自由选择智能体类型，旨在极大提升开发效率，甚至还能配置专用GitHub账户让智能体自动创建PR。
