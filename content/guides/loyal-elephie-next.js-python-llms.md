## Loyal Elephie

Loyal Elephie 是一个结合了 Next.js 前端和强大的 Python 后端的 AI 项目，利用最新的大语言模型 (LLMs) 和检索增强生成 (RAG) 技术，提供无缝且有意义的聊天体验。以下是其主要特点和使用场景：

## 功能特点

1. **🎛️ 可控记忆：** 用户可以控制 Loyal Elephie 的记忆，决定保存哪些对话内容，并随时编辑上下文。这使其成为用户的第二大脑，用于存储事件记忆。

2. **🔍 混合搜索：** 结合了 Chroma 和 BM25 的强大搜索功能，优化了日期查询。

3. **🔒 安全的网络访问：** 内建登录功能，确保只有授权用户可以访问 AI 伴侣，保证对话的隐私和安全。

4. **🤖 精简的 LLM 代理：** 使用 XML 语法，无需调用函数，优化了缓存生成，支持在 Llama.cpp 或 ExllamaV2 上运行的本地 LLM。

5. **📝 Markdown 编辑器集成（可选）：** 可以与在线 Markdown 编辑器连接，在聊天时查看原始文档，并在编辑笔记后实时整合 LLM 知识。


## 使用场景

1. **个人助手：** Loyal Elephie 可以作为一个个人助手，帮助管理和召回重要的事件记忆、任务和笔记。

2. **教育工具：** 可用于教育领域，帮助学生在学习过程中保存和检索重要的学习资料和笔记。

3. **商业智能：** 在企业环境中，可用于保存和检索会议记录、项目进展等信息，促进团队协作和知识共享。

4. **医疗场景：** 医生可以使用其记忆和检索功能，管理和回顾患者的病历信息，提高诊疗效率。

5. **开发者工具：** 开发者可以将其用于开发笔记管理和代码片段保存，提升开发效率。


Loyal Elephie 支持多种本地和专有的大语言模型和嵌入，作为 OpenAI 兼容的 API 使用，提供灵活的部署和应用场景。

**注意：** 该项目原本设计为在 **Linux** 上运行，对 Windows 或 macOS 的兼容性尚未完全测试，使用过程中可能需要自行解决一些兼容性问题。
