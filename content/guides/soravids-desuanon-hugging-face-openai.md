## SoraVids

SoraVids 是一个由 desuAnon 发布在 Hugging Face 平台上，旨在提供 OpenAI 视频生成模型 Sora 的生成视频及其相关提示。该项目发布了 87 个视频（约 702 MB）和 83 个对应的提示，这些内容是基于临时获得的 OpenAI API 访问权限生成的。

在发布后的几个小时内，OpenAI 撤销了该 API 密钥，导致无法再生成新视频。发布的所有视频均为 MP4 格式，帧率为 30 FPS，一些生成的提示在生成过程中可能会被修改，因此实际使用的提示可能与列出的不完全相同。

SoraVids 的内容被标记为 CC0 1.0 公共领域，旨在用于教育、艺术和研究目的。项目不隶属于 OpenAI，且原始视频都有 OpenAI 的水印以显示内容来源。
