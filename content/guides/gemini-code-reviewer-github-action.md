## Gemini AI Code Reviewer

**Gemini AI Code Reviewer 概述**

Gemini AI Code Reviewer 是一个 GitHub Action，利用谷歌的 Gemini AI 自动审查拉取请求（PR）。它通过 Gemini API 提供代码审查的反馈，包括对代码的评论和改进建议。

**主要功能：**

* 自动审查 PR，提供代码修改的反馈。
* 针对代码提出建议，帮助开发者改进代码质量。


**使用场景：**

1. **代码审查自动化** ：在团队协作中，开发者可以使用该工具来快速获取代码审查意见，提高审查效率。
2. **改进代码质量** ：通过 Gemini AI 提供的反馈，帮助开发者发现潜在的代码问题及改进点。
3. **促进学习** ：初级开发者可以通过审查的建议，学习更好的代码编写实践，提升技能。
4. **减轻审查负担** ：自动化的代码审查减少了团队成员之间的反复沟通，帮助项目更迅速推进。


此外，设置过程简单明了，用户只需获取 Gemini API 密钥并进行配置，就可轻松开始使用。只需在拉取请求中评论 `/gemini-review`，即可触发审查流程。
