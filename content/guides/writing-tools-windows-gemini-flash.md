## Writing Tools

##### Writing Tools 简介

Writing Tools 是一款受苹果智能应用启发的 Windows 应用程序，旨在通过 AI 提升用户的写作能力。它是全球首款系统级的智能语法助手，允许用户通过简单的快捷键修正语法错误和其他写作问题。

##### 选择 Writing Tools 的理由

* **智能 AI** ：基于谷歌的 Gemini 1.5 Flash 模型，效果显著优于许多本地模型。
* **系统级功能** ：可以在任何支持文本选择的应用程序中即时使用，且不会干扰剪贴板。
* **完全免费和开源** ：无订阅费用，无隐藏成本。
* **隐私保护** ：用户的 API 密钥和配置文件仅存储在本地设备上，没有日志或广告，并且仅在用户指令下才会发送数据。
* **多语言支持** ：可翻译多种语言，并基于上下文智能处理文本。
* **用户友好的界面** ：直观美观的模糊渐变设计。
* **可定制** ：用户可以设置快捷键以便快速访问。


##### 使用场景

1. **校对** ：快速检查和修正语法与拼写错误。
2. **改写** ：为文本重新措辞以提高可读性和影响力。
3. **调整语气** ：将文本调节为更友好或专业的语气。
4. **摘要** ：为较长的文本创建简洁的摘要。
5. **提取关键点** ：突出文本中的重要信息。
6. **创建表格** ：将文本数据转化为结构化表格。
7. **自定义指令** ：根据特定需求为文本修改提供指令。


##### 使用方法

1. 在任何应用程序中选择文本。
2. 按下快捷键（默认：Ctrl+Space）。
3. 从弹出菜单中选择一个选项或输入自定义指令。
4. 观察文本的提升效果！


##### 总结

Writing Tools 是一个强大的写作助手，适合需要进行快速校对、调整或翻译文本的用户。其易用性与智能功能能够满足多样化的写作需求，使用户在各种场景下都能大幅提升写作效率。
