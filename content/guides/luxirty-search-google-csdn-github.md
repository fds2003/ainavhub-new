## Luxirty Search

Luxirty Search 是一个基于 Google 的搜索引擎，目标是提供干净、简洁、快速且无干扰的搜索体验。 它通过内置内容农场屏蔽列表（如 CSDN、华为云等）以及广告和跟踪链接移除，来优化搜索结果质量。 此外，它还提供一键提高特定网站（如 GitHub、Stackoverflow）权重的功能，方便程序员搜索技术信息。

**使用场景：**

* **需要高质量搜索结果：** 当用户希望避免内容农场和 SEO 优化过度的内容，获取更可靠和有价值的搜索结果时，可以使用 Luxirty Search。
* **程序员搜索技术信息：** 通过一键提高 GitHub、Stackoverflow 等代码相关网站的权重，可以快速找到需要的编程资料。
* **注重隐私的用户：** 由于屏蔽了广告和跟踪链接，Luxirty Search 可以提供更注重隐私的搜索体验。
* **需要自定义搜索结果排序的用户：** 通过手动添加黑名单或优化名单，用户可以根据自己的需求定制搜索结果。
* **追求简洁和无广告的搜索体验：** Luxirty Search 界面简洁，没有广告干扰，适合对搜索体验有要求的用户。
