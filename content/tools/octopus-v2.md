Octopus v2 是一个拥有 20 亿参数的开源语言模型，专为 Android API 量身定制，能够在智能手机、汽车、个人电脑等端侧运行。与 GPT-4 相比，Octopus v2 在准确性和延迟方面表现更好，并将上下文长度减少了 95%。此外，Octopus v2 模型还比 Llama7B + RAG 方案快 36 倍。这个模型在推理速度和准确率上都表现出色，特别适用于边缘计算设备。
